Koha Library Management System Released

pudge on 2002-07-11T11:04:28

pate writes "Koha, a freely redistributable application for managing libraries (the kind with books in them), has just released version 1.2.1 of their software. Koha provides tools to support the back-end work that happens in libraries (circulation, reserves, item acquisition, etc.) as well as patron access (a web-based card catalog)."

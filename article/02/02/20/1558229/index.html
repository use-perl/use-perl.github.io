perl-i18n Mailing List

KM on 2002-02-20T15:52:29

With the help of Ask (perl.org's mail admin) we have created a new mailing list to discuss using Perl for i18n (Internationalization). If you are interested in sharing ideas, experiences and answering questions about using Perl to accomplish i18n, be sure to send mail to perl-i18n-subscribe@perl.org.

List archives will be at http://archive.develooper.com/perl-i18n@perl.org/

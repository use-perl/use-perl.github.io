#news on irc.perl.org - The Perl News Channel

rjbs on 2006-07-12T03:59:00

jbisbee writes "I'd like to formally announce Perl-related news aggregation in #news on irc.perl.org. The channel has been up and running for a while, but after YAPC::NA 2006, I replaced all my code with Plagger and a YAML config file.

Currently rssbot watches the following Perl sites and notifies the channel on updates

  • CPAN (search.cpan.org, not PAUSE uploads)
  • AnnoCPAN
  • CPAN Ratings
  • use Perl
  • use Perl Journals
  • Perl Monks (Newest Nodes)
  • Perl Jobs
  • The Perl Foundation
  • The Perl Foundation News
  • Perl.com
  • The Perl NOC
  • Perlcast
  • The Perl Summer of Code 2006
  • and Slashdot :)

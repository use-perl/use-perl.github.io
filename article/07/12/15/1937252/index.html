Strawberry Perl 5.10.0 - The last big development push

brian_d_foy on 2007-12-13T19:55:00

Perl::Dist (the toolkit used to create Win32 Perl distributions in the Vanilla Perl family) is now dynamically generating it's own Inno Setup build scripts as it assembles the distribution components.

This means I'm now very very close to getting installers proper working.

I'd very much like to have this working by the end of the weekend, so I can give the gift of "complete platform equality" for Perl's 20th birthday (even it it's only a release candidate).

And of course, I also want to achieve a suitably awesome "First Distribution" status, with a release of Strawberry Perl available for download within about an hour of the official release tarball being uploaded. :)

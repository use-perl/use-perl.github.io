New MacPerl Sites, Lists Introduced

pudge on 1999-03-03T13:47:47

The MacPerl Pages have moved to www.macperl.com, and www.macperl.org has been set up for the MacPerl Community. Both sites are run by Prime Time Freeware, publishers of the book MacPerl: Power and Ease.

The macperl.org domain is currently being used primarily for new MacPerl mailing lists. macperl and macperl-announce have replaced the old lists of the same name on a different server (the subscriber lists have been moved and the old address is forwarding to the new). new lists include macperl-porters, macperl-toolbox, macperl-modules, macperl-webcgi, macperl-anyperl (cross-platform Perl), and macperl-forum (anything vaguely related to Macs, Perl, Apple, or comuputing in general).

To subscribe to any of these lists, send mail to <listname>-request@macperl.org with the word "subcribe" in the message body, or use the subscription form.

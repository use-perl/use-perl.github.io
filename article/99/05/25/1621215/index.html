BBEdit 5.1 Adds MacPerl Integration

pudge on 1999-05-25T16:21:15

The latest version of the popular Mac OS text editor BBEdit, released today by Bare Bones Software, comes with a MacPerl menu that allows tight integration with the MacPerl application. Among other features, it allows running scripts from within BBEdit, direct opening and saving droplets and other binary forms of Perl scripts, and access to Shuck (the POD reader that comes with MacPerl). Also included is a filter mechanism to run Perl scripts on the contents of a window, and the ability to call Perl scripts from HTML files during processing of them.

New Perl Books Due for Release

pudge on 1999-10-06T08:48:39

Manning Publications, publisher of Damian Conway's Object Oriented Perl, has released Elements of Programming with Perl by Andrew L. Johnson. The book introduces programming using Perl, and assumes no prior programming experience.

Tuomas Lukka's Object-Oriented Programming in Perl was due to be published in September 1999, according to amazon.com, but according to the author and Apress, the publisher, it is due to be released in February 2000.

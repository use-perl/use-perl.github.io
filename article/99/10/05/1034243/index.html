iPlanet Web Server to Bundle Velocigen

pudge on 1999-10-05T10:34:43

A lite version Binary Evolution's VelociGen is being bundled with iPlanet Web Server Enterprise Edition 4.0 (formerly called Netscape Enterprise Server) from Sun-Netscape Alliance. VelociGen is an application server for Perl and Tcl programs, similar in concept to using Apache with mod_perl.

The lite version of VelociGen bundled with iPlanet is functionally equivalent to the full version, but allows only a single Perl processing engine. The full version allows a user-defined number of engines, and allows distributing the processing over several machines. The lite version can be converted to the full version with an activation key.

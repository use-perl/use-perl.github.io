Perl 5.6 Heads Toward Beta; Development Release 5.005_62 Posted

pudge on 1999-10-15T16:25:20

Gurusamy Sarathy released perl 5.005_62 today, the latest development release for perl 5.6. Included in _62 are support for our() (which declares lexically scoped global variables), the addition of File::Glob (so perl can do globbing without invoking a shell), an optional warning for concatenating "19" with another number (a potential Y2K issue), more 64-bit enhancements, subroutine attribute declarations, bug fixes, enhancements, doc changes and additions, and other interesting things.

Sarathy said that he is working on getting out a beta from now on, so he will have a "very strong bias and resistance against putting in new features that (he's) not already convinced about."

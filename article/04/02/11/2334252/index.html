Defined-or Patch for perl-5.8.3 Revised

pudge on 2004-02-12T01:04:00

merijn writes "Shortly after Nicholas released perl-5.8.3 to the world, I uploaded the defined-or patch to CPAN, as is described in perl581delta. For 5.8.3, the defined-or patches were uploaded to CPAN as authors/id/H/HM/HMBRAND/dor-5.8.3.diff, but this patch had one tiny off-by-one glich, that didn't cause perl -V to show that the patch was applied, even though everything else was fine.

Now that PAUSE does not allow to upload the same file again, I had to delete the old patch file, and replace it with one with a slightly different name: authors/id/H/HM/HMBRAND/dor-5.8.3a.diff. As the old patch file has been removed, and we - from perl5-porters - expect people that want this patch both know how to fetch it and how to apply it, we are confident that they are able to find it under it's new name. Enjoy, Have FUN! H.Merijn"

Source code for searchable CPAN database

Simon on 2004-03-08T23:07:00

rkobes writes " As part of an update of our site for searching CPAN at http://cpan.uwinnipeg.ca/htdocs/cpan-search.html , I've repackaged things so that it should be installable on other machines. The (alpha) source code is available from the CPAN-Search-Lite project on SourceForge. This doesn't have all the great features of search.cpan.org, but it may be useful to those interested in setting up a minimal searchable CPAN database - it can even be used on a box without a local CPAN mirror (albeit without the capabilities of extracting module and package docs).

The documentation for setting things up is minimal - please contact me, or use the SourceForge mailing list, if you have questions, run into some bugs, or have suggestions." Mmm, code. Thanks, Randy!

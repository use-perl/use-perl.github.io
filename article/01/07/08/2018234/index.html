ActiveState Releases

pudge on 2001-07-08T20:12:37

ActiveState has been hard at work as usual, and has released the July 2001 edition of ActiveCD, which includes ActivePerl, ActivePython, ActiveTcl, source code, the Perl Package Repository, Python Packagre Repository, Komodo, Perl Dev Kit, PerlEx, and 21-day trials of Visual Perl, Visual Python, and Visual XSLT.

Phew.

The CD is $39.95, plus shipping and tax.

Visual Perl is a plugin to Microsoft's Visual Studio.NET, which includes the Regular Expression debugger and other IDE features. It is now in beta 4 release.

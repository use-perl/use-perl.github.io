Free Service: PDFmail 2.0

pudge on 2001-01-15T22:31:43

SANFACE Software writes "PDFmail is a free Iservice, developed using perl.
You can use it via our 3 standard forms at SANFACE Software site: HTML interface, WML interface (you need a WAP phone or a WAP browser to use it) and i-mode interface (with japanese support) or you can modify these interfaces without limitations to use the PDFmail service from your web. Please, don't use improperly this free service. Here you can find a complete documentation."

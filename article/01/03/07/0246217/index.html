PAUSE upload tool

ziggy on 2001-03-07T02:43:21

hfb writes "Neil Bowers posted the following announcement to the module-authors mailing list regarding his new version of the cpan-upload script. which makes uploading code to PAUSE even easier. Thanks to Neil and Andreas!"

I've just uploaded cpan-upload 1.8 to CPAN:

file: $CPAN/authors/id/N/NE/NEILB/cpan-upload-1.8.tar.gz
size: 6229 bytes
md5: e361c37db3c427cd40b695367687b518

cpan-upload just automates the process of uploading something to CPAN via PAUSE:

  • upload the file(s) to the pause ftp site
  • POST to the pause web site to register the upload

For example, to upload this file I ran:

% cpan-upload cpan-upload-1.8.tar.gz

Thanks to Andreas for prompting me to sort this out and release a working version.

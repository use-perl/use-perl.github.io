Binary Distributions of Perl 5.6.1-TRIAL2

pudge on 2001-02-13T15:03:15

jand writes "We have released ActivePerl build 624, which is based on the Perl 5.6.1-TRIAL2 sources.

Please note that this is a TRIAL release to give you an early chance to check if your code has been broken by the patches for 5.6.1 (there are several hundreds of changes). Please do not use it on production systems.

The perldelta.pod document has not been updated in 5.6.1-TRIAL2; the only information about changes is the "Changes" file in the source distribution. But approximately 95% of the changes are bug fixes anyways. Please report bugs and incompatibilities between 5.6 and 5.6.1 to our bug database. We'll either resolve or forward any issues as appropriate.

Jan Dubois & the ActivePerl team"

More on RSS

pudge on 2001-05-10T12:05:30

eisen, the author of XML::RSS and co-author of the RSS 1.0 spec, writes "I recently finished re-writing Weblog and have released it on Mother of Perl. Weblog is an experimental platform for implementing RSS 1.0 applications. Weblog manages one or more RSS channels. New items can be added to Xhoo (zoo), a Web directory, using the RSS taxo module. I also developed a set of Weblog modules that make it easier to manage multiple RSS files, which will support an XML-RPC interface in the next release. You can play with a live implementation @ http://www.textant.com/weblog/bin/weblog."

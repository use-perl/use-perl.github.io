Perl on NetWare

pudge on 2001-08-01T19:36:47

Ananth Kesari writes in that perl 5.6.1 (with a patch) and perl 5.7.2 build on NetWare. It includes a UCS (Universal Component System) interface for accessing NetWare resources, including NDS (Novell Directory Services). The source is included with perl 5.7.2, and more information (and binaries) are available at http://developer.novell.com/ndk/perl5.htm.

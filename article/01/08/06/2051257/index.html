CSS list and project for Perl

pudge on 2001-08-06T20:51:17

Robin Berjon writes "A new project has been started to put in common and coordinate CSS (Cascading Style Sheets) related developments in the Perl community. The project is hosted on SourceForge and available from http://sf.net/projects/perl-css. A mailing list (perl-css-devel@lists.sourceforge.net) has been created to go with it, you may subscribe to it from http://lists.sourceforge.net/lists/listinfo/perl-c ss-devel. Anyone is welcome to join to help with the development."

Bravo!

perl.org lists

ask on 2001-08-07T23:09:37

If you are doing a Perl related list and don't like mailman (or majordomo or whatever), then you are always welcome to write to ask@perl.org to get a perl.org mailinglist setup.

Included are web archives, nntp server and superior spam filtering. :-)

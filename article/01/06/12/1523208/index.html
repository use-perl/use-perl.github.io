Repository Browser and Other Goodness

pudge on 2001-06-12T15:24:25

Simon writes "There's been a bunch of fun things this week (and it's only Tuesday! Woo!) which maybe helpful for Perl developers. First off, the Perl repository browser had been jazzed up and is now hosted at ActiveState where the repository lives; this should be a lot faster - thanks to Gurusamy Sarathy for setting that up.
Thanks again to Sarathy and also to Ask, you can get notification of changes to the repository via the new mailing list Perl 5 Changes.
Finally, I hacked around on Sunday and produced a hypertext Perl grammar generated from the Perl sources."

All very cool. I love the repository browser. I am skeptical that I will love the changes list (let's join YA mailing list right before leaving on a trip!), but I will subscribe anyway.

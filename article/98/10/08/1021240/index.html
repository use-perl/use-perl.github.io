MacPerl and Perl for Win32 Top Download List

pudge on 1998-10-08T10:21:40

We aren't sure how long MacPerl has been tops at DOWNLOAD.COM in the Developer Tools category, but we know that it has been at least two weeks running, and about two months ago, it was number two. So it has probably been up there for awhile now.

And we know that MacPerl is STILL not one of DOWNLOAD.COM's Top Picks (selections of best available software) in the category, but cgi-lib.pl 2.17 still is. :-)

Also odd is that Perl for Win32 is tops on the PC list. In and of itself this is expected, and not odd; but the fact that it has only be on the list for one week is, indeed, odd. It's the 5.004_02 build, and it is also listed as a Top Pick in its category, after only one week. ActivePerl (which is a package from ActiveState containing Perl for Win32, Perl for ISAPI, and other things) has apparently been on the charts for some time, too.

Do these numbers have any meaning? Well, it seems some inferences can be drawn -- but not too many -- from the fact that Perl is topping development tools charts on both Mac OS and Windows. The numbers might matter to a good portion of pointy-haired bosses out there, so we suppose they are worthwhile to that end.

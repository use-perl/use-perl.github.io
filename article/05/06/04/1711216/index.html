Perl 5.8.7 released

rafael on 2005-06-04T17:29:00

Nicholas writes: "The Perl 5 developer team is pleased to announce the Perl Release 5.8.7, the seventh maintenance release of Perl 5.8.

The CPAN ftp multiplexor will pick a mirror close to you, if you select either bz2 (9839086 bytes) or gz (12512211 bytes) here; or, you can choose your own from one of the 263 CPAN mirrors worldwide."

Read details of the announcement below.

md5sums for the tarballs:
9a175d6ccbb5d9b41ffac5073ff9cc3c perl-5.8.7.tar.bz2
dacefa1fe3c5b6d7bbc334ad94826131 perl-5.8.7.tar.gz

5.8.7 is a maintenance release for perl 5.8, incorporating various minor bugfixes and optimisations. Please see the perldelta for the full details. Please report bugs using the perlbug utility. If the build or regression tests fail, make nok. If the build fails to early to run this, please mail perlbug at perl.org directly.

This is a source code release, not a binary release. You will need a C development environment to build the sources. Binary releases will be made available by various vendors. To build and install Perl, and to find out how to report problems, please read the INSTALL file, and any relevant README.platform file. As specified in the licenses for Perl (see the files named Artistic or Copying), THIS PACKAGE IS PROVIDED WITH ABSOLUTELY NO WARRANTY. As always, you should conduct an appropriate level of testing before using any new product in your production environment.

The perldelta, which describes the most important changes for this release is available in the source distribution. You may also be interested Perl 5.8.6 announcement, and can find out much more about Perl in general at the Perl home page.

-- Nicholas Clark, on behalf of the Perl5 Porters, 2005-06-03

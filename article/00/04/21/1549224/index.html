Blind Perl Programmers

pudge on 2000-04-21T15:49:02

Abigail writes: Blind people use computers a lot, and blind people can be found in many computer related fields. There are blind news admins, system administrators, and blind programmers. I have, however, never heard of a blind Perl programmer. I'd like to know whether Perl is more blind friendly than other languages. On the one hand, Perl uses a lot of different punctuation symbols, which makes it awkward to read when a speech program is used. On the other hand, Perl code is relatively dense, which would help if you are using Braille reader.

Are there any blind Perl programmers than can shed a light on this?

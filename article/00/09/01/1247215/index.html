ActivePython Beta Released

pudge on 2000-09-01T12:46:52

Yes, we know it isn't Perl, but ActiveState, everyone's favorite Perl On Windows company, after moving into the Linux and Solaris market with ActivePerl, is now in the Python business with the first beta release of ActivePython, also for Windows, Linux, and Solaris. ActivePython 1.6, build 100 is based on Python 1.6b1.

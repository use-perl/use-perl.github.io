Free Software Trusted With Important Data?

pudge on 2000-08-20T13:59:35

havoclad writes "Slashdot featured the New Linux Myth Dispeller, but of interest to Perl, is FAQ 4.12 Production Data can't be trusted to free software. The main example to dispel this 'Myth' is that everyone trusts perl, and perl is free, so linux must be ok as well."

Well, close ... it's not that Linux is OK because it is free as Perl is, but Linux shouldn't be excluded just because it is free, since Perl is free, and it is trusted.

Running Grants: biweekly report

brian_d_foy on 2008-08-01T21:07:00

Starting this month, the Grant Committee will publish reports about running grants each two weeks (first and third Friday of the month). While it is normal that not all grants have updates (there are a lot of reasons for that to happen), they will appear in the listing as well with a 'no updates' message.

Currently running TPF grants are:

  • Mango - No updates.
  • Porting PyYAML to Perl - No updates.
  • Improving Smolder - Grant is finished. A final report should be published very soon.
  • Perl on a Stick - A first release of Strawberry Perl Portable Edition for Mobile Devices and Flash Drives was done. This same release was offered in Flash disks during OSCON.
  • SMOP - Grantee did not start yet.
  • Fixing Bugs in the Archive::Zip Perl Module - Grantee did not start yet.
  • Make localtime() and gmtime() work past 2038 - Started. No other news.
  • Test::Builder 2 - Started. No other news.

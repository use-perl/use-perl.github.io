Testing at chicago.pm

pudge on 2003-05-14T12:47:37

inkdroid writes "At the next chicago.pm meeting, on May 19, Andy Lester will be speaking about the hows and whys of creating automated tests for your code and projects. Andy is the current maintainer of Test::Harness, and is going to be presenting 'Automated Testing of Large Projects With Perl' at the O'Reilly Open Source Conference in July. So if you're not able to make it to Portland stop by, have a drink, and learn about the Zen of Perl testing. O'Reilly has been kind enough to donate a few more books to give away at the meeting (thanks Marsee): Google Hacks, Managing and Using MySQL, Apache, the Definitive Guide. 3rd ed., MySQL Cookbook, Sendmail. 3rd ed."

chicago.pm 2003.04.21

KM on 2003-04-11T01:06:25

inkdroid writes "At the next chicago.pm meeting (04/21) there will be a short tech session on SOAP. If you come you can expect to learn what SOAP is, why it might be useful to you, and how to create SOAP clients and servers using Paul Kulchenko's lovely SOAP::Lite.

More importantly, Marsee at O'Reilly has donated some books to chicago.pm which are going to be given away:

  • Programming Web Services With Perl
  • Embedding Perl in HTML with Mason
  • Linux Server Hacks
  • Perl Pocket Reference
  • Perl CD Bookshelf, V3.0

More info is available at chicago.pm.org"

Perl 5.8.2 RC2 is out

Simon on 2003-11-03T22:17:00

nicholas writes "Perl 5.8.2 Release Candidate 2 has been uploaded to CPAN. I hadn't planned on RC2, but there have been a few significant tweaks since RC1, most notably in some library calls with threading. See the summary on perl5-porters or read the perldelta. Download the source from http://www.cpan.org/authors/id/N/NW/NWCLARK/perl-5.8.2-RC2.tar.bz2 (or as .tar.gz ) Please promptly report any problems as detailed in the INSTALL file, section 'Reporting Problems.' RC1 will soon be heading for a happy retirement on backpan - may the Schwartz be with you! "

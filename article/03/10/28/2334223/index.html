Perl 5.8.2 RC1 is out

pudge on 2003-10-28T23:49:00

nicholas writes "Perl 5.8.2 Release Candidate 1 has been uploaded to CPAN. 5.8.2 is being released to fix minor binary incompatibilities discovered between 5.8.1 and 5.8.0 in the hashing code. 5.8.2 should be binary compatible with both, so if you have modules already installed with 5.8.0 or 5.8.1 please check that they work with 5.8.2. RC1 shipped without a perldelta (D'oh!), so it is on CPAN separately. Download the source from http://www.cpan.org/authors/id/N/NW/NWCLARK/perl-5.8.2-RC1.tar.bz2 (or as .tar.gz) Please report any problems as detailed in the INSTALL file, section 'Reporting Problems.'"

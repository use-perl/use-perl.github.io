While the Editors have god-like powers and can add karma points, upvotes and downvotes, which affect Karma, are typically added by Moderators. While the Slashdot system is very slightly different than that used on Use Perl (for example, Karma has a numeric value displayed on Use Perl still, on Slashdot it's a class), there is a page that explains most of it on Slashdot.

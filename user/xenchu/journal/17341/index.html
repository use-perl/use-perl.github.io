Karma...

jordan on 2004-02-11T01:50:48

Is pretty meaningless here, and nearly meaningless on Slashdot, where it (and the code that runs this site) originated.

The idea is that when you've demonstrated enough good posting, that you aren't just some wicked troll or something, you'll get to post with a "Karma bonus" that'll give your posts an automatic +1 rating. I've got the Karma bonus, so I could always post at 2, but I have it turned off and only apply it when I think I've definitely said something worthwhile.

This can be important on Slashdot where you don't really want to read the postings at 0 or (shudder) -1. You might not want to waste your time on 1s. I've browsed Slashdot at 3 or even 4 so as to not waste time with the crap and me too comments.

As I said, all this is pretty meaningless here, but this place, amoung other things, is pudge's playground for the Slashcode, so we have all the features of Slashdot, even one's that don't make much sense here.

Zig

chaoticset on 2005-02-05T20:46:35

While I realize something like this probably already exists on the CPAN, I figured it wouldn't hurt me to get my hands dirty and build a few wheels, so I wrote Google::Zig*, which just measures the estimated count of a list of queries on Google and returns it.

This was specifically requested by someone on a LJ comm as part of a stats thing they're doing, but they didn't know how to do automated Googling. I fired off an email to them about how they'd like it packaged. I'm not sure how much module installation they've done, and I'm not sure how well I can package the module since I've never packaged a module for installation before.

Even if they don't want it for some reason, though, I'm trying to figure out some useful things I could do with it myself.

* Zeitgeist Immaculata Google. Don't ask.

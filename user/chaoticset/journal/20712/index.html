Trillian has Jabber support. But it costs $25, IIRC. Basically, you have to buy Pro, which supports plugins, and there is a Jabber plugin available. Hmm, maybe Basic 3.0 will make Jabber support available to all.

The one thing that rules out Miranda for me is that the Yahoo plugin is not upto scratch. At one point, it didn't work with proxies, it still has quirks, and I have a lot of contacts on Yahoo. Other than that, it's a great client (with an unbelievable set of plugins).

It's not necessarily better than Gaim, though. A smaller initial download and it seems a lot more polished than Gaim on Win32 (maybe I just don't like the Gtk widgets on Win32)

chaoticset on 2001-12-10T17:09:27

I decided I should keep a journal of the stuff I'm doing on the 486, so I wrote a tiny little script to handle the journal entries for me. Due to a sudden fit of nostalgic reading and a deranged mind, I decided to format it like an old-time Shadowland* document, like this(pretend the parenthesis are appropriate angle brackets):

((Date, Day, Time))

)))))[The body of the entry is here.](((((

This took a little bit of work, because just writing it directly to the linuxjournal file didn't make it come out right. Eventually I got to the point where I built an array out of all the lines to go into the journal, then iterated over that array while each last line was in a buffer variable. If the buffer was the )))))[ starter, then no linebreak was printed after it. If the current line was FINAL (my signal to the script that I'm done with entering the entry), then it adds a ]((((( without putting a linebreak after the last line. Otherwise, it just prints the line with a linebreak, just like I stored it in the array.

It took me maybe thirty minutes to work all the little kinks out. (If I ever get ambitious I'll try to make it word wrap properly.)

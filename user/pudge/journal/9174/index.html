Tabbing in a form

KM on 2002-12-02T17:57:10

I haven't tried 1.2 yet, since people keep saying it is hosed (so, still with 1.2a). Try adding this to your user.js file, and restart Mozilla. See if it helps with your tabbing in forms (works fine in 1.2a):
// Enable tabbing to all fields in a form,
// as well as links.
user_pref("accessibility.tabfocus", 7) ;
That file should be in:

~/Library -> Mozilla -> Profiles -> LOGIN -> RANDOM.slt

(LOGIN is your profile name, RANDOM is random stuff). If it doesn't exist, create it.

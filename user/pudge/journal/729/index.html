Even More Errors

pudge on 2001-09-01T03:49:32

Well, CHECK isn't soon enough to catch compile time warnings and errors. So Mac::err is an XS pragma now. -MMac::err=unix reverts to Unix-style errors. The pragma can be used anywhere in the code now, but in code the function would be preferred.

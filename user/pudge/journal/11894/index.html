Images in MP3s

pudge on 2003-04-28T21:43:37

iTunes 4 can store album cover art in the MP3 tag.

perl -MMP3::Info -e ' $file = shift; $x = get_mp3tag($file, 2, 2); ($i = $x->{PIC}) =~ s/^(....).//; ($e = $1) =~ s/\W//g; $file =~ s/\.mp3/.$e/; open $f, ">", $file; print $f $i; ' song.mp3

This can extract it!

Major Panther Bugfix

pudge on 2003-11-27T16:46:05

Has anyone else noticed that file sharing works a lot better under Mac OS X 10.3? I have not had any problems unmounting file servers. Before, I often could not unmount because the Finder was busy doing stuff, so I would have to force quit the Finder. Or if I slept and awoke, I could not unmount at all, and accessing it might freeze up everything. Panther fixes all this.

I typically don't have iPhoto open when I insert a memory card but when I insert one, the Image Capture app launches. I don't have Panther in front of me (or any version of iPhoto) but in Jaguar the Image Capture preferences includes:

When a camera is connected, open:
iPhoto
Image Capture
No application
Other

I don't think these preferences have changed in Panther. Perhaps if you set this to something other than iPhoto, iPhoto won't import. I would have thought iPhoto would have its own import preference but I guess if it did, you wouldn't be posting.

Note

pudge on 2003-07-29T20:27:55

High-bit Latin-1 chars will be converted to the appropriate character reference, unless there is an appropriate ASCII version hardcoded in the code (like 'curly quotes' -> 'straight quotes' ... suck it up!).

In Extrans and Code modes (and ECODE tags), which do not allow any character references, those character references will turn into literal text, like &#171;, because why would you use that character in Code anyway? Silly.

Website Meta Language 2.0.11 is Out

Shlomi Fish on 2006-08-19T07:02:29

Website Meta Language version 2.0.11 is out. This is a bug fix release that corrects the build on cygwin, and also integrates some vendor patches. It should appear on Freshmeat.net soon.

WML is a sophisticated offline hypertext processor, written in C and Perl. 2.0.11 is the first announced release in a long time. The Roadmap I posted to the mailing list contains what is planned for it next.

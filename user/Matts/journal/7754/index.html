I don't know whose laptop is this, but it can't be that painful to have some *nix on it. And if you can, having two unices on it: Linux and FreeBSD.

If you lived nearby, I'd be pleased on installing Linux or FreeBSD for you, but we are something like 12000 km from each other :-)

BTW, if you already know FreeBSD, I've seen that it has very good Perl support and made my laptop fly. Due to its memory management system, I think that its performance is higher than Linux on laptops.

If you don't know, try getting an ISO from Conectiva. Just set the correct PCMCIA type on your laptop's BIOS and you should get everything running out of the box.

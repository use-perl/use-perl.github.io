More Lyx

Matts on 2003-07-06T18:34:23

Well the Lyx On Aqua project finally works. Yay! I have a working Lyx installation.

Installation wasn't what I'd call trivial - certainly not exactly by Mac OS X standards, but still, at least it seems to all work.

The binary on the web site when I installed it missed out the entire /usr/local/share/lyx directory, which was a bit of a showstopper for me, but I managed to download the lyx source, and compile and install just that directory without any problems. After that, I made sure all my fink stuff was up to date (including tetex), edited my Environment.plist, logged out and in again and I had a working Lyx.

This is fantastic. On Linux it replaced OpenOffice as my document writing tool, much to the chagrin of my co-workers (who hated getting PDFs they couldn't edit - so I mailed them the Lyx files ;-)

Now I'm beginning to wonder how fast we'll see other Qt/KDE apps starting to appear native on OS X (i.e. without running X11). I'm hoping Kate, the KDE text editor will appear sometime soon...

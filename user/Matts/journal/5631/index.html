Why Pod?

Matts on 2002-06-13T14:12:28

In case you're wondering why I've been hacking Pod::SAX, I've been scratching in itch...

For a long time now I've felt two things:

  1. AxKit lacked a decent simple example of an easy to use, easy to write, and funky web app.
  2. Perl lacked a Wiki that supported POD syntax.
So I wrote one.

Yes, it looks butt ugly right now. But the look is all in XSLT stylesheets, so that can change. Also the formatting is done using whatever SAX parser you might have - so it's quite possible that I'll extend it in the near future with support for ordinary Wiki text once we get a SAX parser for Wiki text (which Jörg Walter is working on).

Anyway, please feel free to play around with the Wiki, though expect me to delete the whole lot at some point and start over.

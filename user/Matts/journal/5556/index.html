AxKit 1.6

Matts on 2002-06-10T20:27:42

After what seemed like an endless run of release candidates, AxKit 1.6 (final) is now available here.

Release notes (or Changes) are available here.

There's a lot of really cool features in this release. I think that finally AxKit can be considered a proper component system now - with a much more flexible system for including separately processed components into pages. And it's finally seen the separation of content and style providers, allowing you to build neat things like POD providers, or Email providers (or the OpenOffice provider, which I'll be selling soon).

Thanks to everyone who made this release possible, notably Kip Hampton and Jörg Walter who did a lot of the coding.

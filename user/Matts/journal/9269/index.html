Mac OS X users might like to try Searchling - a small unobtrusive menu item that pops up a search dialog for different sites when you click on it. It even remembers the last place you searched.

I've extended it by modifying the XML so it can do search.cpan.org searches too. Let me know if you want a copy of my XML file.

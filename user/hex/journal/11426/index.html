Wikibot 0.3

hex on 2003-04-03T15:21:50

As of now Wikibot 0.3 is available! There are two big differences: first, it now uses POE::Component::IRC instead of Net::IRC. This fixes a strange timeout problem it had and is generally a good thing anyway (thanks to acme for tips on how to get that working). Secondly, you can now tell the bot what page on the wiki you want to add to. This is a major usefulness upgrade. Hooray!

The only instance of it I know of is my one running in #core on irc.rhizomatic.net, which adds things to my new wiki project, accretion. If you find it useful, please let me know!

Figured out some bookmarklets to follow site navigation links by adapting code from Subotniks guide

 LE = document.getElementsByTagName('LINK');
  for (i = 0; i < LE.length; i++) {
      if (LE[i].rel == 'prev') document.location = LE[i].href;
  }

Added them to the Galeon menu of my remote, so now I can present stuff with navigation links easily. It is no coincidence that podblog already generates them.

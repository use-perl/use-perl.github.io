Writing HTML

godoy on 2002-08-21T20:59:21

It's amazing the dependency that tools like FrontPage cause.

My brother was creating a website and he was using FrontPage.

  1. He got amazed to know that there is an HTML validator
  2. He could use elements such as "li" and "ul" to get an itemized list even with CSS (he said he wasn't able to get it with FrontPage)


It made me think while I was at the WC:

Why these tools are so "I do what you want to see, not what you want to do"?


FP knows what he wants to see. It doesn't know --- and doesn't care with --- the information he wants to transmit to users.

Such tools should create good code. Should create CSS files with the document (even if it used CSS inside the document, in it's beginning, it would be better than spreading "font", "span" and other elements all around).

Is it so hard to write good and valid HTML?

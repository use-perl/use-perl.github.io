WikiWiki systems and UI design.

godoy on 2002-01-12T21:21:51

Several months ago I had to choose a WikiWiki system to use at home.

I've reviewed some and I was discussing at #modperl (on irc.rhizomatic.net) why do Perl programmers don't care more with the UI?

Take a look at TWiki and MoinMoin. TWiki is a Perl application (it would be better if it was a mod_perl one) and Moin is a Python application.

I chose Moin. Why? It was more pleasant to use and to read. What you see at my home is it's default installation. TWiki's default is not pretty. Is not pleasant.

So, Perl programmers, please, worry more with the UI. I know we can do better. We can get more users and programmers to the “light side of the force” if we have more pleasant things.

New ARM

ziggy on 2002-04-30T13:25:02

The ARM folks are at it again. In a press release yesterday, they detail the new ARM11 micro architecture, which sounds quite impressive:

  • 1200 MIPS, peak
  • SIMD extensions, DSP extensions
  • The 16-bit THUMB instructions are still in there for ultimate code density :-)
  • Standard RISC features (integer pipelines, out-of-order execution, branch prediction, etc.)
  • Still a low power CPU
  • 64-bit memory system
  • 70% reduction in interrupt latency possible
  • Accellerated context switching
Um, wow. When can I get that 64-way ARM box to replace that dual-Celeron on my desk? :-)

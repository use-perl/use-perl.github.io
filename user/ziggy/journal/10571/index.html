Software Updates

ziggy on 2003-02-13T20:06:35

Apple has released an update to Safari via softwareupdate. It does exactly what it says it does: update Safari. (I deleted the previous version of Safari from ~/Applications, and the update "succeeded" without actually updating anything. I always download software update packages before applying them, so I re-installed the update after I re-installed the previous version of Safari in /Applications.)

Apple's other recent update is version 0.2.1 of their rootless X Server. This fixes quite a few bugs from the 0.1 release. X windows minimize properly now; before, they could minimize once, but subsequent minimizations wouldn't display an icon in the dock. Also, input is harmonized with the standard drivers; switching keyboard configurations also works within X apps. Previously, switching keyboard configurations would only work with Mac apps, but X apps would continue to use the keyboard config they started with.

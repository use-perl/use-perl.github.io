svn 1.2.0

ziggy on 2005-05-24T20:46:01

Today, I had occasion to build a new shared SVN repository at $WORK. It's been a while since I built subversion, and there have been some updates in the iterim. The new fsfs backing store has been added and is now the default (no more worries about BerkeleyDB wonkiness, if that was a concern).

Turns out, my currently installed svn client is a year and a week old now: 

$ svn --version
svn, version 1.0.2 (r9423)
   compiled May 17 2004, 15:13:56

Copyright (C) 2000-2004 CollabNet.
...and version 1.2.0 came out yesterday. ;-)

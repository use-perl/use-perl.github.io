WiFi Mesh

ziggy on 2003-06-10T00:27:46

Want a wifi network throughout your entire home? Want to get rid of the dead zones in the house?

Paul Boutin says the only consumer wifi access point that interconnects easily into a mesh is Apple's Airport Extreme. :-) Start with one, and add extra base stations to extend coverage into the dead zones.

(Of course, you could move into a townhouse without wire mesh stucco walls instead, where pretty much any base station will cover the entire house. :-)

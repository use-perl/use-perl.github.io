How NOT to install your new NAS

jk2addict on 2007-03-12T13:35:05

This is a quick how-to-not guide on installing your new DNS-323 RAID1 NAS product.

  1. Put drives in unit and plug it into the network
  2. Turn power on
  3. Upgrade Firmware
  4. Assign an IP address already in use somewhere else
  5. Disable ICMP in Windows Firewall Settings just for your wireless card connection

#4, and #5 are really critical if you want your random file copy errors, network not found errors and file in use errors to work properly.

Without these steps, you may end up with a normally functioning NAS unit on the network.

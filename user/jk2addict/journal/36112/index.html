Tidy, Applesauce, Quarter Rolls and Purple Shrouds

jk2addict on 2008-04-09T17:30:14

I feel like I've just joined a cult, but at least this one doesn't require that I bring toll for the passing alien ship, or eat my apple sauce with cyanide or cover up on my special purple shroud.

As perrin points out, it's about consistency. Once I convince myself to stop worrying about how the code looks, and more about how the code works, all will be well...and consistent looking.

With that said, and a critic test later, my .perltidyrc file is now:

-l=78
-ce


That's it. Cuddled elses are a must, and sometimes, tidy makes things with 81 characters. 78 seems to fix those cases.

Test Everything: Part 4 of x

jk2addict on 2006-10-24T12:51:49

Yesterday, the ability to connect to excel documents and import them appeared to be b0rKen on the database server. Somewhere between the last service pack and now, connecting to excel files just tossed up a quite unhelpful error. After some tinkering with temp directory permissions and restarting the server, all is well.

But, the next time (and there's always a next time), I'd like to know about this a lot sooner. Now I have a test for it.

So far, the non-dist related tests are:

  1. Test part images exists on servers
  2. Test names in table X for duplicates
  3. Test product page contains only requested products data
  4. Test that required params are getting sent in pagination links
  5. Test for successful connect and select from excel doc within db

Perl6: A better documentation resource

markjugg on 2006-08-24T13:00:14

Yesterday I worked on improving the Perl6 documentation. Agentz had created a shiny version of the Perl6 spec that could display tests related to each section inline. Very nice!

I helped to improve the front page for this system, providing a table that also references the justification (Apocalypse) and explanation (Exegesis) for each part of the spec, where available.

Also collaborating with agentz, we brought Perl6::Perl5::Differences online as an HTML, linked to from the same page.

The Differences doc is still very much a work in progress. It needs to be extended, organized, and further cross referenced with the spec so readers can easily jump to more detail. With Agentz's smartlinking script, it's fairly easy to create links to the spec from there, too.

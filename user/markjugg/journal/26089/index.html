HTML::Template::Plugin::Dot is born!

markjugg on 2005-08-04T14:33:18

I refined my addition of TT's magic dot notation to HTML::Template. It has now been re-cast as the first plugin for HTML::Template.

Of course, that meant adding a plugin system to HTML::Template first..

It's rough now, and I'm looking for collaborators to help refine it.

You can read my post to the HTML::Template mailing list for all the details, including a link to the patch, and a darcs repo for the project.

The code is in my darcs archive.

CGI::App Link Integrity plugin released / HtmlTidy news

markjugg on 2005-08-19T19:37:10

Michael Graham has released the lastest CGI::Application plugin: CGI::Application::Plugin::LinkIntegrity

. This plugin ads a checksum to URLs which allows detection of changes to them, preventing certain kinds of tampering. The action taken when a tampered URL is noticed is configurable.

In other CGI::App news, Andy Lester has released a new stable version of HTML::Tidy. The HtmlTidy plugin depended on some functionality in the last developer release of HTML::Tidy, so this new release will make the plugin easier to install and use.

Thanks to Andy for the release and Rhesa for the plugin!

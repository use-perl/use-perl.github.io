Catalyst summary for 2005/04/30-2005/05/02

draven on 2005-05-03T09:31:48

Hey kids.

Sorry about the lack of updates this weekend, but really, it's been sorta quiet, and I've spent most of my time either outdoors or painting my office, so time has been scarce.

I did manage to squeeze in time for a interview with Perlcast tho. It was included in Perlcast #7. Go check it out for a good laught at my expence. :)

There's also been some CPAN activity. On saturday we pushed some needed fixes for existing modules. The CDBI model was updated to require the latest CDBI-Loader, to let the user see any error messages on initialization, which had been supressed before. The Browser & Unicode plugins also needed some bug fixes, one my fault, the other chansen's :). Finally, we released a updated Catalyst::Model::CDBI::Sweet, in preparation for moving the core to a stand alone Class::DBI module.

I also found time on Sunday to package up and release Chansen's Geography plugin. It's pretty sweet, lets you get country/code from the user's ip.

Omega finally found some time to put his EasyCMS repo online. This is another nice Catalyst example. He's got a demo online here, but no public login so far. I've gotten a peek tho, and have to say I'm impressed, just need a designer's touch, and the html looks quite sane, so that should be piece of cake. Gabb, are you up for the challenge? :)

Some more views are coming today, you can check the repo if you are curious. They are all packaged up, just need to be pushed out.

Last night, when they should be in their beds, Sri and Chansen had a long discussion about reusable application components. There's a reusable article something coming up, once we finish the discussion on how to name everything :)

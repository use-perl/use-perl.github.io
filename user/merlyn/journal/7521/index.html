Far too geeky to watch a movie

merlyn on 2002-09-04T09:56:42

OK, so I'm watching Antitrust because my TiVo decided to record it, and I hadn't seen it before. I see HTML scrawl by during the opening credits. I freeze frame a bit of that, and quickly get disinterested. But at 24 minutes into the show, I see some C or maybe Java code scrawl by. I freeze frame it, and recognize it as some sort of HTTP widget (parsing for GET and HEAD). I pick out a couple of the juicy keywords, hit Google, and there the code is, in all its glory.

No, I wish I was making this up.

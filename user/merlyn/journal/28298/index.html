Easy - use CGI.pm

SuperCruncher on 2006-01-10T16:18:59

Use the CGI module:
use CGI;

my $cgi = CGI->new();

my $foo_param = $cgi->param('foo');
my $bar_param = $cgi->param{'bar');
Obviously CGI.pm can do a lot more, but that's enough to get you started.

CPAN Testers Update

Ovid on 2009-05-26T08:28:06

Tomorrow there should be a new release of the Test::Simple distribution. You can check out the changes at Schwern's github Test::More repository. The only known breakage was my Test::Aggregate module, but that's understandable because I am doing things which Test::Builder isn't designed to support. So after a bit of work to ensure that Test::Aggregate worked with old and new versions of Test::Builder, I went out to the CPAN testers page for my distribution. I am very pleasantly surprised with the new update. I particularly like the easy OS/Perl Version matrix. The Perl community should be proud. I would love to see other language communities step up to the plate like this.

Test::Harness::Straps is going away soon!

Ovid on 2007-10-05T09:13:22

On the TAP::Parser development list, there's been some concern expressed that not enough people know that Test::Harness::Straps is going away with the upcoming release of Test::Harness 3.0. We've been talking about this for months and those who do serious testing generally seem to be aware of this impending change, but I want to reiterate this for those who don't know.

THS is fundamentally broken and those who've developed and maintained it agree that it has to die. It was a hack to get around design problems in Test::Harness. To understand how easy it is to code a replacement, check out the developer versions of Test::Harness and chromatic's TAP::Parser review.

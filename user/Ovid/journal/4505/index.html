Feeling like a real programmer

Ovid on 2002-04-28T03:58:35

I just uploaded Sub::NamedParams to the CPAN. It allows you to use named arguments with any subroutine (or specify a new name with the subroutine, if necessary). It was interesting how many times I needed to disable strict and warnings to get it working without screaming at me. I'm almost beginning to feel like a real programmer. Interestingly, this has coincided with my work. We've finally started to get over a "development stagnation" and I'm actually learning new stuff again. This is a relief.

In other news, my brother is coming next month, I'm going to have a boarder for three weeks, and I mysteriously found myself captain of our pool team, though I am certainly not the best player and have only been playing on the team for about three months. This is roughly how I wound up running Portland Perl Mongers: no one else wanted to do it.

Idle thought: to get the aforementioned module to work, I had to use typeglobs. Thought I've been trying to pay attention to Perl6, I can't help but wonder how this will work without typeglobs.

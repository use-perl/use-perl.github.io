Oslo Perl QA Hackathon Letdown

Ovid on 2008-04-09T09:17:15

First off, the hackathon was fantastic. Linpro treated us very well (and they're hiring) and the space for us was perfect. However, the one thing I really, really wanted out of the hackathon was YAML diagnotics emitted from Test::Builder.

We're not going to get it. Schwern tried, but without Test::Builder 2.0, it's not architecturally feasible. That's a shame because I've written most of the code necessary to handle them in Test::Harness.

We could start using them tomorrow if Test::Builder could handle it :/

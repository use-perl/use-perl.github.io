Class::CGI 0.14 and Class::CGI::DateTime 0.01 released

Ovid on 2006-04-25T23:26:26

Thanks to a bit of discussion with radiantmatrix, I've removed Class::CGI's dependency on Module::Load::Conditional and released 0.014. Now it runs just fine with taint checking.

I've also released Class::CGI::DateTime. This module uses all of the features of Class::CGI and is a good example of how to exploit its power. At this point, I think I may be able to list Class::CGI as beta instead of alpha code.

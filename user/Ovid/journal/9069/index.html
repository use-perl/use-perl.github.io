Downgrading

Ovid on 2002-11-22T00:21:44

Stayed late at work last night. We had a Web site that we finally got a chance to upgrade after having a client put off our migration for almost a year. The migration was fairly smooth, but we ran into one little snag. The staging server uses M$ SQL Server 2000, but it's 7.0 on the production box. Microsoft apparently does not like downgrades. After finally getting that to work, the site still wouldn't come up. We were getting mysterious error messages from Template Toolkit, but upgrading from 2.04 to 2.08 solved the problem.

TGE added to Parrot

Allison on 2005-11-12T02:38:41

This week's Parrot day went to checking the tree grammar engine (TGE) into the Parrot repository. Okay, it wasn't entirely spent on the commit, it also went to refactoring the code for sanity, beefing up the documentation, setting up some of the standard framework pieces (a t/harness script, auto-generated Makefile, etc), and converting my scripts that exercise TGE's features to tests (Parrot has a problem running tests on libraries outside the repository). So, if you go looking for it, the latest version of TGE is now in the Parrot repository at compilers/tge.

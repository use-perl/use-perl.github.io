Backport of Data::Dumper for 5.6.1 is finally out ..

IlyaM on 2003-07-20T19:01:04

.. and it took only 1 year. 

The URL

    http://martynov.org/tgz/Data-Dumper-2.12_01.tar.gz

has entered CPAN as

  file: $CPAN/authors/id/I/IL/ILYAM/Data-Dumper-2.12_01.tar.gz
  size: 37406 bytes
   md5: ca00bd288a917a6e0cddc936853ddf99
It passes all tests on 5.8.0 and blead perl and passes all tests on 5.6.1 with exception for some UTF-8 tests. In this regard pure perl version of Data::Dumper is completly broken on 5.6.x and it is probably not worth trying to fix as Unicode support is broken in 5.6.x anyway. Other failing tests are because 5.6.x cannot store Unicode strings in hash keys. If nobody finds any other problems with this release on 5.6.x and 5.8.0 I'll release 2.13 soon.

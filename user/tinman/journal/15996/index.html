Mail clients and IMAP

tinman on 2003-11-25T10:39:14

Had to move mail clients from Pegasus to Thunderbird. Didnt like that too much (had to be dragged kicking and screaming away from Pegasus really. Amazing how attached I got to that piece of software in 4-5 years of use). But Pegasus mail appeared to be somewhat dodgy on IMAP so I didnt have much choice. Had to manually create all the filtering rulesets, though, which sucked a bit.

Getting used to Thunderbird now. Just get a moderate amount of spam for the moment, which I can delete myself, so the junk mail is turned off. The client freezes at times, especially on a slow link, but hey, its only 0.3 Even if it gets no more features, it would still be a pretty good mail client.

Installed XFCE 4.0.1 on my Mandrake machine. I dig minimalistic desktops :)

GraphViz::DBI

TeeJay on 2003-03-29T17:49:30

Well I just uploaded GraphViz::DBI version 0.02 to CPAN, only a couple of weeks after Marcel handed over.

The documentation and code is all neat and tidy and lovelly and I'm rather reluctant to turn it into a mess.

I put in a couple of small changes so that it works similar to the code in Autodia for doing the same thing (i.e. using the record shape, left aligning fields, finding Primary Vs Foreign keys better).

Now it runs correctly on the test databases I have used for autodia, so I'm happy and plan to feed changes to/from autodia and graphiz::dbi as I go along.

new autodia coming soon

TeeJay on 2003-02-23T23:37:22

I know I *do* on about autodia but a) people seem to be interested (about 100 emails and close to 12,000 downloads) and b) its my only significant open source project. (VCG.pm and a rather redundant search engine are neither very popular or very large).

anyway...
I have just been polishing it a little and I have stashed a (probably broken) pre-release of 1.4 at the site in the downloads directory. This pre-release includes some nice new functionality

  • outputing in gif/jpg or pbm/ps depending on the name of the destination file when using Graphviz or VCG
  • VCG support (basic, many features to be added)
  • DBI handler (experimental)
There have also been some improvements in how it does PHP and C++ handling.

You can find some newer images created with improved handlers and the new VCG code in the images/ directory on the site.

It needs some testing before I release properly - the dbi handler could do with some work and the code is looking a little scruffy again, I also need to add better support for perls object orientation.

I hope to release a nice polished release that doesn't throw errors when converting from pod to man and supports SQL as a language some time in early march.

hope you guys like it.

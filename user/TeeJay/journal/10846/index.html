TeeJay on 2003-03-01T18:03:07

I installed blosxom on our CVS / bugzilla / file server at work on friday. Its another of those simple lightweight things that do exactly what they are supposed to, no more, no less.

It doesn't need a database, or any modules, and it worked first time. Cool!

I quickly hacked a new header and footer and story template, and added some test storys and categories. I have a feeling I may add some bizarre hacks to it, or make it use TT, but not yet..
for my first trick I will add a cvs_to_blog perl script to our CVS servers loginfo which will add files blosxoms data directory so they just magically appear in the blog.
Then I will hack bugzilla to add files into the data directory each time a bug is added or closed.
Hey presto, a nice integrated intranet!!

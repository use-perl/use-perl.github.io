autodia 1.6

TeeJay on 2003-03-30T17:37:53

Thanks to Sam Tregar for pointing out the total absence of quality control in Autodia 1.5 (well just the SQL handler, the rest all works as normal).

I overwrote the SQL handler late one night and lost about 30 minutes work and being tired (hence the overwrite in the first place) I released without fixing it.

The make test (which ran fine) didn't catch the problem because it doesn't test any of the handlers. So now I have added a minimal test suite. This should also make resolving installation problems somewhat easier.

Anyway version 1.6 of autodia is at the site and winging its way towards your nearest CPAN mirror.

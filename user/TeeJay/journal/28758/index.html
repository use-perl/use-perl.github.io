maypole 2.11 progress

TeeJay on 2006-02-21T17:49:28

Just to let anybody interested know, we are now close to releasing Maypole 2.11.

I have it passing all but a handful of tests and with only 2 or 3 outstanding bugs in rt.

Some key benefits of this release include

* Silent death should be relegated to the past * New Debug information pages on error when debug flag is set. * Even smarter factory forms * Integrated sub-pages/components without plugins * many bug fixes * much more documentation * Mime guessing fallback if you're too lazy to set a MIME-Type for a page * lots more

so Faster, Better, Easier and more reliable..

and only um.. 3 months late!

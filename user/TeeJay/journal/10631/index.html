disappointed with cvsmonitor

TeeJay on 2003-02-17T17:03:29

I tried out cvsmonitor this afternoon and was very disappointed - the user interface and features are great... its just a shame I can't get it to work.

The installation was OK, a cpan bundle made life a little easier, and set up through the web was easy enough.

The only problem is that it insists on using pserver - a real pain if you don't already use pserver - and once I got pserver up and running its still wouldn't work.

The error logging is useless and it doesn't recover at all from a corrupted file.

unable to checkout the single module in which we do all our work I will probably end up hand crafting some code into bugzilla to duplicate some of cvsmonitors features.

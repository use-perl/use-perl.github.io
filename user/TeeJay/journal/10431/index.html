VCG is quite cool.

I eventually managed to install it (by reading the instructions instead of just expecting make, make install to work), gnu autoconf or makemaker would make the install much easier, I had to manually specify loads of files and look around for libs.

The grammar is very similar to dot, but with a bit more structure, easy to parse and create which is nice.

The X interface is a bit rough, and I ought to really read the documentation, but I think it will be a useful format to read and write with autodia.

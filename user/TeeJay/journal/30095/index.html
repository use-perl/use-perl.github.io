Oops - it's been a year since I released Maypole 2.10, since then several developers including myself have been implementing the ambitious road map I set out after releasing 2.10.

On the plus side, 2.11 has been pretty much ready to roll for a couple of months and I've been using it solidly for a couple of inhouse projects. You can download the pre-releases from CPAN and I should have the final pre-release ready this weekend

Anyway.. while some of us have been working on polishing a mature framework - two more new perl MVC frameworks have sprung up :

  • Sri's Mojo framework forked/inspired by Catalsyt (can't find a website for it yet)
  • The Solstice Framework from Washington Uni in the states.

Work found me again

VSarkiss on 2003-12-18T22:20:20

I've been looking for contract work the last couple of weeks, but in kind of a "low-impact" mode. I figured I wouldn't find anything before the beginning of the year, and so far I've been right.

Last night I got an email from an old friend who's taken a CIO position at $BIG_mortage_bank. I don't know any details about the project, but it looks like it's going to involve data warehousing with MS SQL Server. It's very familiar territory, even though it's not a lot of fun. But I'm sure I'll end up using Perl for something -- so far, Perl has crept into every data conversion or data warehouse project I've worked on.

And the best part, of course, is having income to pay the Xmas bills.

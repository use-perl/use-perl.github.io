Operating system confusion

VSarkiss on 2004-07-31T15:45:05

For the last few weeks, I've been using Fedora Core 2 at work (and at home for the last few months). Last week I had to set up a Windows laptop to use at my client so I could connect to their systems and databases. No big deal: been there, done that. After I had what I absolutely needed, I started installing the apps I usually use on the laptop: Firefox with my favorite extensions, Vim, Gaim, etc.

Yesterday a strange thing happened: While using the laptop, I highlighted a section of text in one Firefox tab, and middle-clicked to paste it in another. I was completely surprised when it didn't work! It took a second to dawn on me that I wasn't using X -- I needed to use Ctrl+C/Ctrl+V to copy-and-paste.

It hadn't really sunk in how many applications have become cross-platform. Or maybe it's just the early stages of Alzheimer's.... ;-)

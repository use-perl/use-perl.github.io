HighEdWebDev Conference

cbrandtbuffalo on 2004-10-10T18:31:58

I'm currently at the HighEdWebDev conference being held in Rochester, NY, this year. This conference used to be WebDevShare and used to be held at IU each year. So far things seem cool with the new location, although many people are bemoaning the inability to go to Nick's for a beer.



Being a higher ed conference, open source is fairly well represented. I'm doing my best this year to show that perl isn't just for cgi scripts anymore. I think we'll have some perl luminaries here this year, specifically an appearance by brian d foy.

Open Source Going Mainstream?

cbrandtbuffalo on 2005-02-03T21:19:54

Talk about worlds colliding! I got my Gartner Application Integration and Web Services brochure today. I've attended this conference before, so I'm on their mailing list. Not a bad conference, but they are all about Java and .Net. Open source and Perl don't really exist for Gartner and their analysts.



So you can imagine my surprise when I got my brochure and it announced one of the speakers is none other than Tim O'Reilly! Are the Gartner folks finally 'getting it?' It seems they might be, but I'd love to know the story behind this. Maybe brian can get the scoop from Tim at OSCON this year. Maybe sooner for The Perl Review.

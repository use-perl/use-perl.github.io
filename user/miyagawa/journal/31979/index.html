Catalyst::View::JSON 0.14 is released

miyagawa on 2006-12-21T01:39:34

Catalyst::View::JSON 0.14 is released and is going to CPAN. Here's the Changelog.

- Added UTF-8 BOM if the User-Agent is Safari and encoding is utf-8 (Jun Kuriyama) - Added 'no_x_json_header' option to disable auto-setting X-JSON for Prototype.js (Jun Kuriyama)

It's backward compatible and if you don't have problems with Safari nor Prototype.js you don't need to upgrade.

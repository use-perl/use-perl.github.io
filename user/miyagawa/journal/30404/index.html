Hope this helps.
The newer version of the Win32::Gui.pm has a ShowBalloon method and also you can pass params
-balloon => 0/1
         -balloon_tip => STRING
         -balloon_title => STRING
         -balloon_icon  => STRING
        -balloon_timeout => NUMBER

As per the docs here

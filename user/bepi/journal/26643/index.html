[ANNOUNCE] POD2::IT 0.09

bepi on 2005-09-08T09:36:38

Hi all,

the pod2it team (Italian translation project of core Perl pods) is proud to announce a new POD2::IT release available on CPAN.

CHANGES FROM PREVIOUS VERSION

  • + Added perlunicode.pod translation [dakkar]
  • + Added resources/podenc.pl script in order to encode italian characters with E<> sequence [dakkar, bepi]
  • ! Updated E<> pod entities [bepi]
  • ! Updated IT.pm pod documentation [bepi]
  • ! Fixed Changes and README typos [bepi]
POD2::IT also contains the patch to Pod/Perldoc.pm in order to continue to use -f and -q switches with translated pods (not only italian) with the syntax:

%> perldoc -L IT <podname>
%> perldoc -L IT -f <function>
%> perldoc -L IT -q <FAQregex>

I added a -L switch that allow to define language code for desired language translation. If POD2::<code> package doesn't exists, the effect of the switch will be ignored.

I hope that a p5p guy will apply it to Pod::Perldoc... :-)

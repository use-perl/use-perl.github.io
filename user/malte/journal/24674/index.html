New major mod_perl site

malte on 2005-05-13T20:43:27

Today, we relaunched http://www.lottospielen-nds.de/ The Lotto-Platform for Lower Saxony in Germany.

Both front-end servers and app-servers run mod_perl and they talk to each other via RPC::XML. Everything runs quity smoothly. The performace is really remarkable. Our average load today was under 0.1 with a Lotto Jackpot oder 14 million Euros (which is quite high and makes people crazy :)

Next week we'll launch the (probably) first Lottery with live internet broadcasting of the draws every 3 minutes. The client application is a Flash app that talks to the server via XML requests (think Ajax). Upon fetching the current drawing numbers we syncronize the client time with the server and then let the client figure out the time of the next draw and display a countdown until then. It actually works quite well, but we'll see.

AryanResistance.pm

jmerelo on 2002-09-22T19:15:50

How would you feel if there was an AryanNation.pm group in your country? Imagine what I felt when I found a ResistenciaAria.pm group in Spain. Just in case the translation is not clear, take a look at the web page of the same name (which happens to be geographically situated in the same region, so I don't have many doubts the PM group is created by the same people as the web site).

I have started a campaign asking for that group to be removed, or, if that is not possible, that its name be changed to something else, like Vigo.pm (the city it seems to be based in). I'm asking all of you who feel the same to write to the webmaster of pm.org website.

Purdy on 2001-11-07T11:04:14

Posted this in the PerlMonks' Meditations:

I found this neat-o Windows code editor: Crimson Editor. It has color coding and all that, but the additional feature of setting up FTP sessions and opening/saving files remotely.

Then the Perlmonks kindly told me that the remote file feature is now run-of-the-mill (I guess no one told Komodo that) and to take it to the next degree, emacs supports this as well as editing them over SCP/SSH.

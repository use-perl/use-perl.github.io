Call for Ideas...

Purdy on 2002-05-15T17:03:16

I was watching one of our graphic artists' productivity grind to a halt and made me wonder if there would be any way to automate her task (to convert individual Quark Express pages to PDF). Here's what she does:

  1. Hit 'Print' from Quark (and make sure the printer's filter is selected)
  2. Type in the individual page number
  3. Hit the 'Printer' button, which brings up the 'Create PDF Printer' dialog.
  4. Name the file 'NNNqsrPP.pdf' (where 'NNN' is the page number and 'PP' is the issue number)

This then feeds the Quark file into the Spooler, which 'chews' on the file and converts it to PDF. She additionally has to check the filesize of the resulting PDF when it's complete, to make sure something valid came through on the other end (12Kb usually means it's just the column marks and no content).

Does anyone have any ideas? This is on a Mac [OS 9], btw.

Skype me!

jouke on 2004-11-24T10:56:35

I'm not always the first to try out new software. Obviously I'm not the first one to discover Skype, but since a colleague of mine pointed out that it could be a cheap alternative for making many calls a day, I thought I'd give it a try...

Now I've been trying quite a number of internet telephony software in the past years, but none of them satisfied me...either there were too few users, or it was hard to install, hard to get a good connection, whatever. This time I downloaded, installed and used the software and I haven't found anything yet that I don't like about it. Actually, just one thing....Not everyone is using Skype!

There are Windows, Linux and MacOS X versions, so most people should be able to use it...and if you installed it: Skype me!

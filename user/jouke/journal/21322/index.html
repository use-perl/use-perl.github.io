ImageMagick works! :)

jouke on 2004-10-13T16:04:20

All credit goes to Jean-Louis Morel, the maintainer of this PPM repository. He extended his Image::Magick patch to make it work on Windows 98. I tested it, and ImageMagick (and therefore the latest development snapshot of pVoice) works on Windows 98/ME/NT4/XP. Unfortunately it still doesn't work on Windows 95, but the Magick.dll doesn't load at all there, so that's a different problem all together.

I don't mind that it doesn't run on Win95 anymore. Windows 98 can be installed on all machines that run Windows 95 (I think). People will just have to upgrade...

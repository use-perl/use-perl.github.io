Time for a new beta?

jouke on 2004-10-29T09:50:30

Sometimes external factors push me to release something although I'm not confident enough to do so. It happens at work, but now it happens with pVoice.

Now there are two important reasons to release a new beta. First of all, Krista needs new functionality that I built already. Secondly, next wednesday a consultant from RDG Kompagne will demonstrate the Mercury computer (see earlier posts in my journal for more on this beautiful piece of hardware) and he would like to show us how pVoice will run on that machine. I'd like to see the bleeding edge version running of course...

Unfortunately I changed quite a lot since the previous beta version: support for more types of speech synthesizers (most notably SAPI5 support), the use of ImageMagick to do the image handling (so support for more image formats) and plugin support. All of these work fine on my own PC, but since it's not tested thoroughly yet, I'm hesitant to release anything of it to the public yet.

Since Krista will be with me this weekend again, and the consultant plans to install the latest version next Tuesday, I either have to start testing tonight, or just postpone the whole thing until I've tested it better... My gut feeling says 'wait a little while longer', but I also want to make pVoice look good for the consultant....not to mention I want to give my daughter what she needs asap...difficult...

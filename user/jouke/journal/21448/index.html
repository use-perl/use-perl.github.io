More on plugins

jouke on 2004-10-21T09:26:17

Finally I have the pVoice codebase tidied up a bit more. I had to do this to be able to get plugins working in pVoice. And yes, yesterday evening I created a first plugin that did nothing more than display 'Hello World', but it was just a proof of concept...

Now that I have that working I can start implementing pStory, pHouse and pMusic as pVoice-plugins. I've been wanting to do this for a long time, but hacking on the same codebase for two years doesn't do much good for the maintainability of it, so the cleanup I did was really nessecary first. When I have the feeling that the Plugin API has been sorted out better (when I implemented at least one not too complex plugin), I'll be releasing a new beta, which will be beta 4 of pVoice 2.2.

If not too many new bugs appear, this will be followed by the release of pVoice 2.2. That would mean a new version before the end of the year...

beta 3 nearly done

jouke on 2004-09-14T09:08:06

The third beta of pVoice 2.2 is nearly done. Actually, I thought it was done yesterday on my way back from work until I discovered that I had introduced a new bug. I can't release a new version (even if it's a beta) when I *know* that there is a real bug.

So, I think I need one more evening of coding (which unfortunately is not going to be tonight...) before releasing this beta.

Boy, so many things have changed this time. Especially the 'edit-mode' has been improved a lot. You can now decide to jump to another theme after selecting a word (for example you can jump to a theme 'nouns' after selecting an item in the theme 'verbs'), you can view the result of what the button will look like real time while editing it, you can decide not to use an image for the button, in which case the caption text will appear in a font as large as possible, centered on the button, etcetera. All of the items on the TODO list for 2.2 regarding the 'edit-mode' are now completed.

I'm really excited about what 2.2 will become. Now I can only hope that the users will be just as excited :)

Wiki and CPAN

jouke on 2002-08-28T12:52:33

This morning I set up a WikiWiki for wxPerl. I was pleasantly surprised by the ease of setting it up. I was afraid I had to do all kinds of difficult things to set it up, but it came down to extracting the files, setting permissions correctly, changing the httpd.conf, editing the wiki.cnf and I was ready to roll.

Now let's hope people will start using it...

And I've registered myself with PAUSE, so in a few days my Win32::MSAgent module will be on CPAN :)

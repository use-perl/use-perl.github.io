All the Perl that's Practical to Extract and Report

of snmp, printers, switches, and a cute miniproject

statico on 2004-05-21T01:32:06

Occasionally I'll stop work on my Class::DBI, Template::Toolkit and SNMP stuff and do something fun for fifteen minutes. Then someone else (John) takes the source code for the fun thing, gives it a third color and even adds the hostname. Presto! A cute little CPU-load bar graph which makes my student group's home page look nifty. I got the idea from sotck.xchng, a nice site for royalty-free images.

Another fun thing was this nify printer statistic thingey. I used scli to generate XML which I stuffed through an XSLT stylesheet to make a pretty page. I really like the miniature console that shows the current message that's on the printer. I think I'll put a bunch of these together and use SoggyOnion to produce and aggregate them.

(I later found out that scli doesn't always produce XML, *and* its text output is intentionally hard-to-parse. Sucks.)

Also, I gave SNMP::Info five stars. It rocks.

Now back to work.

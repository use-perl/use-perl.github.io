Kwiki 0.30 is out - spread the word

mako132 on 2004-06-22T03:00:54

Brian Ingerson has released Kwiki 0.30, the long awaited update to CGI::Kwiki 0.18. Grab it at http://cpan.org/modules/by-authors/id/I/IN/INGY/

You'll need Spoon 0.15 (also on CPAN) this time around.

No upgrade script as yet to upgrade an already existing Kwiki.

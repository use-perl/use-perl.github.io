Moving from GNATS to RT

mako132 on 2002-07-15T14:57:42

I'm moving our low traffic GNATS bug database to RT. I've been using RT for months...but those docs aren't so good. I learn as much be experimenting with a test queue.

We moved to RT because it's easier to customize than GNATS and the web interface is better. The code inside gnatsweb.pl is hairy: too much of the code is based around iterating through the official fields list. I'm sure this was someone's idea of elegance at one point.

Ratpoison

mako132 on 2002-10-23T14:34:43

Via Slashdot I learned of a new ultra-terse X window manager called Ratpoison. This window manager is essentially the same as the text terminal program 'screen'. All your X windows are now full screen and you toggle between them with keystrokes.

This is a big help for me:

  • I'm easily distracted. Keeping emacs in full screen and never moving my hands off the keyboard to flip between windows (typically just other xterms or Mozilla) helps me stay focused. Now I'm not distracted by gabber IM's or the CNN web page.
  • At home I use my tiny Sony Vaio with a 10" screen. I need all that real estate.
Clearly not for everyone. But at work I've switched from Gnome, and at home from Blackbox -- to Ratpoison. Now my nose is to the grindstone.

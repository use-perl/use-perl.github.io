The Truth Behind The Curtain - The Movie

ask on 2003-07-21T17:58:30

The OSCON 2003 movie, The Truth Behind The Curtain is now available online. It has been slightly updated from the version we showed before the last keynote in Portland. We also added an explanation of the jokes. The jokes are so internal that even I had to get some of them explained; so don't feel bad if you want to read along while watching. It's seven minutes, and 50-60MB in the highest quality. (The finished DV file is about 1.6GB, a bit too big to make available :-) ) Nathan has been uploading other goodies as well and more is being added every few days, so go and check it out.

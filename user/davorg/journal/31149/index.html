Dodgy Old Modules

davorg on 2006-09-28T13:41:32

About five years ago, a rather dodgy module called Date::MMDDYY appeared on CPAN. It's sole purpose was to convert dates into the completely illogical MMDDYY date format. And it did that in a bizarrely convoluted way.

At the time I used that code in a course I was running. I asked people to review the code at the start or the course - and then we looked at it again at the end of the course. It was a good way to demonstrate how the students (hopefully) looked at code in a different way having been through the course.

Now there's a chance that I'll be resurrecting that course so I've been digging out all the old materials. But I can't find a copy of Date::MMDDYY anywhere. It was removed from CPAN long ago and it's not even on Backpan.

So I wondered if anyone on use.perl had taken a copy. If you did, then can you please get in touch with me.

Catching Up

davorg on 2001-07-13T12:58:20

Been a while since I wrote anything here, doesn't time fly? So what's been happening...

We had another london.pm social meeting. It all got a bit out of hand. I think this wasn't helped by the people who'd been to Linux Expo in the afternoon. Apparently the Expo was very small and it only took them ten minutes to see everything there. They then spent the rest of the afternoon in the pub.

Had an Iterative meeting this week. The main result was that Iterative are now sponsoring YAPC::Europe - so at least we'll get our name around a bit more.

The Guardian have said they'll be happy to look at any article that I send them about OSCON. I'll probably try to write something based around the Craig Mundie talk.

I (finally) bought myself a laptop to take to the conferences. No I've got about a week to install all the software I want on it.

Finished reading CGI Programming 101. It's not a bad book, but there are some pretty nasty errors in there. I need to get a report back to the author soon. Maybe I'll write that on the flight to San Diego.

DMP went to a second printing. I'd probably be very excited if I knew how big the first printing was.

I seem to have everything organised for TPC and YAPC::E. Looking forward to them both immensely.

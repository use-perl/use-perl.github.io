YAPC::Europe Talks

davorg on 2006-06-01T13:17:52

I finally got round to submitting some talk proposals to YAPC::Europe. The submission deadline is tomorrow, so you'd better get a move on if you're planning to submit something.

I proposed a re-run of the talk on ORM that I gave to london.pm earlier this year, a talk about the programming language ecosystem and Perl's place in it and a tutorial called "Advanced Databases for Beginners" which will be my attempt to stem the flow of people designing databases without knowing how databases work. I hope at least a couple of them get accepted.

And, in other news, I heard that none of my proposals for Euro OSCON were accepted. So if I go this year it'll be as an attendee - which last year proved to be a rather relaxing way to go to a conference :-)

Recruiters

davorg on 2004-06-14T14:27:55

I'm sure everyone reading this has plenty of tales illustrating the stupidity of recruitment agents. This happened to me today.

I get a phone call just before lunch and the agent asks if I'm looking for a contract. When I tell him that I'm not he asks me if I know anyone else who uses Cold Fusion.

It's that word "else" that gets me. The term "cold fusion" has never gone anywhere near my CV. I once had to read some Cold Fusion code as the spec for the project was "make it do what the old system did" but that's the closest I've ever come to writing it. And that's not on my CV.

I guess the agent just thought that Cold Fusion and Perl must be pretty similar as they are both web languages :)

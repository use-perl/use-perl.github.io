Preparation Is Key

davorg on 2002-07-03T13:44:19

Over the weekend I finished the slides for my Idiomatic Perl tutorial. It feels very strange. Usually I'm working on slides right up to the last minute.

I must have forgotten something.

Of course, I've written my own proprietary XML format for presentations. I then use XSLT to convert the XML file into HTML. Then last night I listened to ziggy's YAPC presentation on open source presentation tools and now I'm tempted to redo it all in AxPoint. Perhaps I should just write XSLT to convert my XML to AxPoint's input format. That sounds a lot easier... hmm...

Anyway, whatever format it ends up in, I think it's a good tutorial. If you're going to be in San Diego and you think your Perl could do with some brushing up then, please consider coming along.

Oh, and I'll be giving a sneak preview of some of it at the london.pm technical meeting on July 18th.

Superior technology: svk 1.0

acme on 2005-05-10T20:19:44

Last night we had a small release celebration: clkao released svk 1.0! I offered cake, sky generously offered champagne.

If you're not sure what svk is, shame on you. The blurb goes "svk is a decentralized version control system written in Perl. It uses the Subversion filesystem but provides additional, powerful features". The really cool part is the "decentralized", but quite frankly, the thing that should get you using svk instead of the plain Subversion client is that it's faster and doesn't leave .svn directories lying around everywhere. autrijus uses svk, and see how fast Pugs is being developed - try svk now ;-)

We're having an svk release party in London on Thursday. Obra is even flying in. Come join the party or organise your own svk release party!

Core Perl Modules You Might Not Know About

acme on 2003-01-28T11:38:15

On thursday we had a very successful London.pm technical meeting and of course I gave a talk. Slides for Core Perl Modules You Might Not Know About are now up. Note that I autogenerated these using XML for Keynote, grabbing the title and synopsis from the module itself. Thus, bug reports to p5p please Nicholas ;-)

The talk went pretty well. Most people didn't use the modules I picked. I'm not sure if this is because they work at a different problem-space than these modules or that they simply didn't know about them. I guess we'll see.

Being a pumpkin

acme on 2003-11-28T13:58:35

You know, I thought it'd be really scary being a pumpkin. I mean, the source is scary, and I had to remember how to use Perforce and learn make patches and update the Changes file and so on. But you know, it wasn't that hard.

I've applied two patches by a certain Richard Clamp which makes 5.005_04-tobe compile under recent versions of GCC. Also, I integrated DB_File-1.807 from CPAN so that it copes with newer versions of that library. The current pumpkins are giving me helpful advice, and it looks like I should be able to get a snapshot out soon...

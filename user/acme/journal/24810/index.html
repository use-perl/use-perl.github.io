Net::OpenDHT

acme on 2005-05-21T11:18:52

Distributed hash tables are so Internet 2.0 thesedays. Building, deploying and maintaining a DHT for test purposes is a pain, so Open DHT provides a publicly accessible DHT service hosted on PlanetLab with a very simple interface. Net::OpenDHT provides a simple interface to Open DHT. Still confused? Read OpenDHT: A Shared, Public DHT Service (PDF).

Bonus points for Filesys::Virtual::OpenDHT...

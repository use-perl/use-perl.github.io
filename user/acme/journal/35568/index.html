Blueprint CSS

acme on 2008-02-04T08:03:57

I can't believe I haven't talked about the Blueprint CSS framework yet. Programmers aren't very good at design and CSS is hatefully non-portable. Blueprint takes typographic ideas such as the grid and vertical rhythm and makes them easy to use in CSS. I use this for all my projects and not only is it easier, the results are also prettier.

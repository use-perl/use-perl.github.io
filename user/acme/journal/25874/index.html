Summer of Code

acme on 2005-07-25T10:27:05

The Perl Foundation took part in Google's Summer of Code program and managed to get 8 projects funded. This journal entry is a summary for interviews I conducted about them:

  • A Generational, Thread-Aware GC for Parrot by Alexandre Buisse
  • BitTorrent Peer Library For Perl by Josh McAdams
  • Compiler/Optimizer Improvements to Parrot by Curtis Rawls
  • Extended Type Inference System for Perl 5 by Gary Jackson

  • Open Sourcing Perl/Java Integration by David Rusek
  • Refactor Pirate so that the intermediate code emitter is independent of the Python parse tree (AST) by Curtis Hall
  • Perl Megadistro by David Buchman
  • WWW:Kontent - A Perl 6 Content Management System by Brent Royal-Gordon

HTML::Fraction

acme on 2005-06-19T20:05:20

Fractions? Remember those? 1/3 and so on. Well, my recipes contain them. Turns out, there are HTML entities for common fractions (e.g. &frac12; is ½) and that common vulgar fractions have Unicode characters (e.g. &#8533; is ⅕). ½ looks much prettier than 1/2. Enter HTML::Fraction, which encodes fractions as HTML entities for you! No longer need you wonder how I made the pretty ingredient listing for Banana orange muffins...

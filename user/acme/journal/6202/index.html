Yes, yes, it's true. I've been bitten by the Gentoo bug. So much so, in fact, that I contributed an ebuild for the SDL_gfx library. An ebuild is like a port, it's a magic script that downloads the source code, compiles and installs it. As Gentoo is so bleeding-edge, this means that people can start using it straight away. It kinda feels nice to have contributed something to a distribution and that it be immediately available for all other Gentoo users (well, when they emerge rsync that is).

During the London.pm meeting last week I discovered that I have made everyone addicted to Frozen Bubble. A lot of people turned up and we talked about convex hulls, Buffy, beer, movie scripts and Python. In the middle some of us popped off with the London Perl Cabal to eat at a nice Thai with the current and the dauphin pumpkin, which was great. I dragged them back to the pub just before closing time, where we finished the curiously depleted kitty. Did I say cabal? Errr, there is no cabal. Move along please.

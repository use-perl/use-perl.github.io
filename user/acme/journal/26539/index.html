YAPC::Europe Day 1: Perl In Secure Web Development

acme on 2005-08-31T17:04:17

As the last talk of the first day, Jonathan Worthington presented Perl In Secure Web Development. He started with a bleak outlook (there are millions of holes in software and people just need to exploit one), but then went on to validation (positive validation and taint mode work really well) and protecting from injection. Check out perlsec for more detail. A couple of easy ideas which increase security. Now to hit the town!

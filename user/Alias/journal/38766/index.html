DBD::SQLite 1.20 released!

Alias on 2009-04-07T14:09:29

http://svn.ali.as/cpan/releases/DBD-SQLite-1.20.tar.gz

Since we've run out of RT tickets to fix now, and nothing obvious has appeared in the CPAN Testers results, I've stamped the SQLite driver as "production" and uploaded it so that we can see some feedback from a wider audience.

We've left three more-complex issues to resolve until the next cycle.

* Full 64-bit support.

* Unicode-purity fixes.

* Moving to the Amalgamation SQLite C library.

Class::DBI should work again with this release, and we're a fraction short of restoring full functionality on DBIx::Class, which we can hopefully fix shortly.

Thanks to everyone with commit that have helped over the last week or so to drag DBD::SQLite back into the light.

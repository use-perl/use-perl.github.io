Vanilla Perl 5.10.0 RC1 Build 9

Alias on 2007-11-19T12:03:27

http://ali.as/vanilla-perl-5.10.0-rc1-build-9.zip

I'm happy to announce the first ever (I think) Perl 5.10.0 distribution.

Vanilla Perl is the lead distribution and "experimental" member of the Vanilla Perl familes of Perl distributions.

It provides a pure core-only Perl straight from the tarball, with as few additions as possible other than the C toolchain. In this initial release, that means none.

It's purpose is to allow for the testing Win32 Perl issues, compatibility testing, and package experimentation. It is not appropriate for development, production, or to just "try out" Perl 5.10.0.

This is Build 9 in the series. Vanilla build numbers do not have any significance, or form any order. Quality is (intentionally) highly variable.

Build 9 regresses the .exe installer and reverts to .zip packaging (temporarily). Build 10 is expected to restore .exe functionality.

This release also contains a new directory structure, and removes the pre-built documentation in favour of perldoc.

See the documentation for Perl::Dist::Vanilla 9 (which should appear shortly) for information on installing and setting up the environment for this release.

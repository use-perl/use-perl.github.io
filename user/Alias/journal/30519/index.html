CPAN: The Gathering - Bundle::Email lost, AppConfig gained

Alias on 2006-08-03T02:54:23

Just a note of my latest CPAN: The Gathering (the game where you trade module ownership with other authors) changes.

Bundle::Email has been given away to Ricardo Signes, joining Email::Stuff and unifying both those modules under Perl Email Project control.

As part of Win32 work, I've also taken over AppConfig temporarily to add Win32 support. Once I've cleared out some of the problems, we should then be able to get Template Toolkit installed on Win32.

If anyone wants to take it over permanently, let me know.

As usual, AppConfig is in the Open Repository if anyone wants to fix their own nigglies.

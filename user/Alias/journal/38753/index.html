DBD::SQLite 1.19_08 ready for heavy testing

Alias on 2009-04-04T04:39:51

http://svn.ali.as/cpan/releases/DBD-SQLite-1.19_08.tar.gz

After a whirlwind of patches, RT resolving and general clean ups, I think we've got a reasonable candidate for moving to a production release (pending people slamming the hell out of it).

I expect to see another release or two for further cleaning up of the test suite, but with 40 odd RT tickets resolved, the code itself shouldn't need to change much more (as long as stuff works).

So if you have large or interesting loads, now is the time to throw them at the new releases.

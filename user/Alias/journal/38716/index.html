New (testing) releases of DBD::SQLite

Alias on 2009-03-28T17:54:15

http://svn.ali.as/cpan/releases/DBD-SQLite-1.19_02.tar.gz.

I've recently taken on the task of reinvigorating SQLite development, by taking it into my repository where many people can hack on it and I can just provide release management.

After a fair bit of work, mostly by ISHIGAKI, I think we have a reasonable candidate for a new stable release.

This release updates SQLite to the latest 3.6.11 version (which has support for sub-transactions and other goodies, although some of these may not be available yet from Perl code) and should be reasonably stable.

It passes all the tests for me on both Linux and Win32, which is good enough to throw it out there for wider testing.

Feel free to stress test, benchmark or otherwise fold, spindle and mutilate this release, so that we can move to a production version.

Testing reports in comments to this post would be acceptable.

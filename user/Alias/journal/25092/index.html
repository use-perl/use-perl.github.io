PPI 0.990 - Beta 2

Alias on 2005-06-08T09:54:00

Well, that's the last big step off to CPAN.

PPI now has full latin-1 support in the appropriate places, complete documentation on all 70 classes, and all rt.cpan.org bugs resolved.

It also has a slight change to the constructor APIs to bring all APIs into sync so that they have identical ways of creating things. This might break a few things, particular for anyone using PPI::Document->load, but it is necesary.

I promise it will be the last API change of any significance before release.

In fact, I consider to be the API freeze release.

Have fun!

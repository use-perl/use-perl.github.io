Request for Volunteers: DateTime::Cron::Simple Replacement

Alias on 2006-09-29T06:12:43

This is a public service announcement.

When DateTime::Cron::Simple was uploaded to the CPAN, it contained a copyright assertion but no license of any kind.

Following an enquiry from the Venezuelan Patents and Intellectual Property Office to modules@perl.org on the licensing status, and subsequent talks with the author, the module is being deleted from the CPAN at the author's insistance (for reasons unrelated to the original enquiry).

At least one major Perl application (WebGUI) is known to use this module, and so I'd like to make a request at this time for volunteers to write a legally-clean API-compatible replacement to allow continuity of service for existing users.

Any interested parties should contact, and co-ordinate via, the DateTime mailing list. I will be reassigning namespace permissions appropriately when required.

Template-CTemplate

konobi on 2007-12-10T18:55:47

I've just uploaded a new module to the CPAN Template-CTemplate.

This is currently a fairly raw and basic set of XS bindings to googles ctemplate library. I basically hacked this up to play around with XS and C++ but it seems to have worked out well, so I've uploaded it for others to play with and test.

My C/C++/XS knowledge is pretty limited at the moment, so there's every likelihood that it'll just blow up on any serious usage, but patches are welcome, especially documentation and portability patches.

I'm currently not putting any effort at all into supporting win32, but if someone wants to make it work, lemme know (I'll be targetting OSX, Linux, FreeBSD and Solaris).

Kwiki: another good perl hack

ambs on 2004-09-24T13:50:02

I'm using Kwiki http://kwiki.org and I'm liking it. Simple, stable and expansible. Today installed the first plug-in, to attach files to topic. It worked very nicely.

Congratulations, guys!

BooK on 2004-09-27T12:15:53

For the French Perl Mongers sites, we are using a combination of perl scripts and CVS to update the sites. Interested people have access to the source and templates, and a script updates and builds the site regularly. Commit emails help interested people to know what's happening on the web site.

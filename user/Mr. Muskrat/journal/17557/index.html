MS VC++ .NET Deluxe Learning Ed

Mr. Muskrat on 2004-02-23T03:58:08

I bought a copy of Microsoft Visual C++ .NET Deluxe Learning Edition Version 2003 at Fry's a week or so ago. If you are not familiar with this product, it includes Microsoft Visual C++ .NET 2003 Standard and the book "Microsoft Visual C++ .NET Step By Step" by Julian Templeman and Andy Olsen (published by Microsoft Press if you didn't already figure it out). The list price is $119.99 but I am pretty sure that you can find it for less if you shop around.

I read the book on the train when I am going to and from work. Then I go through the exercised at home when I find the time.

So far I have already known most of it. This should at least be interesting.

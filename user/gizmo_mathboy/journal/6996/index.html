Well, I had a brief amount of joy in life squashed horribly.

My university moved from Apache (I believe) servers to a central web cluster running Zope (at least the schools of engineering that is). I was browsing through The Zope Book and noticed a section about using Perl in Zope.

Well, talking with the web guys I found out that Perl isn't supported because of security issues when using it.

So now I have to climb two hills (Zope and Python) if I want to make good use of Zope for my department. I also want to investigate these security concerns with Perl. I think it's mostly how Zope (Python in reality I believe) handles Perl.

So make that three hills I may have to climb (maybe two and a half if I'm lucky).

Other than that the new job is ok, I think. Always busy, always not really knowing how things should work or even how they could work.

Well, enough lamenting. I have more reading to do before I hit the hay.

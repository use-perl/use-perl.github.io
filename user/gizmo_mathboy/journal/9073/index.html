Linux on the laptp

gizmo_mathboy on 2002-11-22T05:10:27

Well, I finally got around to trying to install linux on my little sr7k. I had tried Mandrake 9.0 but the cd didn't want to boot (burned from downloaded ISO's not sure if that had anything to do with it). So I dug out my purchased RedHat 7.2 discs. Which booted right up.

Only had 2 real bumps in the road: partitioning and grub.

I had trouble getting more than 2 partitions made. This kept me from making a swap partition. No great big deal.

Then, after the install grub only picked up linux. After reading some docs I added in the stuff to boot windows. Worked like a charm.

So, while I have linux installed I think I will still re-install it (make a swap partition and one big partition for the install) and dig around for the drivers for my wireless card and burn them to disc.

Well, that might be enough fun for one night. One small, baby step to minimizing how much I have to use Windows products in my life.

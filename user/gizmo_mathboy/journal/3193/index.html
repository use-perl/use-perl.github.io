At the office

gizmo_mathboy on 2002-02-28T18:43:17

Here I sit in the office configuring my win2k box. It's kind of a downer after using Linux for the last few days.

Those days were spent cleaning up the web and database stuff. It was amazingly painless to upgrade the perl, apache, mod_perl, openssl, and mod_ssl. After that it was just a matter of slogging through the coding and debugging I had to do.

I really need to do a rewrite. It became abundantly clear trying to deal with the the relatively minor database schema change I did. I need to make it much more modular, take advantage of mod_perl efficiencies and persistences, and generally clean the damn code up.

The vpn seems to be working still and I can ssh into the web box from the outside world once I'm vpn'd into the network.

It's a beautiful thing to have a stable, useful network available.

Well, better finish making the win2k box usable (stick cygwin on it so I can have a real shell available :-)

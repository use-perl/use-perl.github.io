Is Resistance Futile?

gizmo_mathboy on 2002-12-17T04:47:23

There are 2 technologies where I work that I'm not entirely sold on: Windows for student lab desktops and Zope for web server/content management.

During a Christmas lunch party I posed the idea that we should be doing more to efficiently use our money (since I work for a state university) and is Windows the most efficient operating system toward that end? One of my coworkers said, "That battle has been fought and lost." Which raised the question in my mind, "Do you stop fighting then?"

Today, I was wondering if Zope is better than Apache. We've moved to Zope because it is a better content management system. It has seem good points with one kicker of an exception, no Perl. I'm currently running a project to convert our current site in Apache to the Zope model.

So the question that is percolating in my brain is whether to be assimilated into what is policy or do I continue to struggle?

My mind and gut say persevere, which I will try to do.

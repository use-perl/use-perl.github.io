Investigating a new mail client

rafael on 2002-07-14T11:29:49

I made my first RPM spec file today. Cool. FWIW it was for the Sylpheed mail client, that I'd like to try. I'm currently using Balsa, which works fine, expect that the threading is a bit flawed. Otherwise, Balsa has some features that I like : POP3 support with multiple accounts, possibility to filter mail through procmail, doesn't wrap your messages if you don't want to. I miss the external editor support (they say it works, but not for me, yet.)

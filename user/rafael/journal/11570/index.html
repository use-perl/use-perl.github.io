I'm now using this script (run from my ~/.Xclients) as a signature generator.

#!/usr/local/bin/perl -ln BEGIN { die <', $sig or die "Can't write to $sig: $!\n"; print $sigfh "\u$u is not *NIX"; close $sigfh; sleep 1; exec $0; } __END__

perlpatch2svn news

rafael on 2003-02-26T12:45:08

perlpatch2svn, my script to import perl patches into a Subversion repository, is now distributed as part of Andreas König's impressive suite of tools Perl-Repository-APC. (I'll delete it from my CPAN directory -- I suppose that's the right way to deal with this kind of situations.) Note the heavily customized Makefile.PL currently distributed in this package. It allows to calculate the version from the subversion revision number, and to propagate and check it at make dist time.

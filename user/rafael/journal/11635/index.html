Chesterton

rafael on 2003-04-14T11:40:01

I spent lots of time, this week-end, in books. I've finished The End of Eternity, and I've read (again) Thomas Browne's Urn Burial. I'm now reading G.K. Chesterton's crime stories from The scandal of Father Brown. If you like Sherlock Holmes, you'll probably like Father Brown. The plot is almost always the same : weird, unexplainable (and criminal) events occur ; Scotland Yard is clueless ; irrational forces are suspected. Father Brown (a catholic priest) comes, and finds a logical solution to the mystery. Includes humorous depiction of characters.

larsen on 2003-11-24T08:03:59

I like to share links and I like the concept of "pre-digested" links (as in digest, and as in digestion :)). Actually, in the italian mongers irc channel #nordest.pm we're using a similar tool thanks to a bot called boha: he can accept URLs, that he puts in a db through a SOAP interface. The same interface can be used by a Perl script or by bohatools' Mozilla sidebar (very useful, since the bot is not always available). More info (and >500 links) here: http://larsen.perlmonk.org/cgi-bin/url.pl.

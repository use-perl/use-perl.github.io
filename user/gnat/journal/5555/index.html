Harry Orsman dead

gnat on 2002-06-10T19:49:33

Who? He compiled the Oxford Dictionary of New Zealand English (a wonderful wonderful book) and was the academic responsible for my university having not only the OED in SGML format, but also the full-text indexing and display tools for working with it. The code I wrote for the OED is still useful, and it's thanks to him that I *really* know about SGML, non-trivial queries in full-text searches, and so on.

He was smart, sweet, and did amazing work. He'll be missed. He already is.

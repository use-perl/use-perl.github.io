Let Down By Technology

gnat on 2002-06-14T17:22:15

The ORA intranet has a page of editorial groups. I wrote a program that builds that page from a simple text file, querying LDAP to get email addresses.

Suddenly it stopped working. Oh, the LDAP server moved. Update the LDAP server location. Still no joy for most people.

Ah. It doesn't recognize their nicknames. Andy Oram isn't Andrew, Chuck Toporek isn't Charles ... It turns out that the nickname-friendly LDAP server was the proof-of-concept. The live LDAP server gets its info from HR, who are not nickname-friendly.

Bugfuck! A beautiful program, spoiled by the real world.

--Nat

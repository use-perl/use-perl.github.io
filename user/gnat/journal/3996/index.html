Firebird

gnat on 2002-04-06T01:37:11

We have an OSCON session on Firebird. Borland released Interbase as open source, but changed its mind later. Freebird is the continued open source development of the Interbase code. The license is a variation on the MPL.

There's a DBI interface to it. It has triggers, stored procedures, and all that jazz. Has anyone here used it?

What really grabbed me about the Firebird project is the website. VERY nicely designed. Extremely fast loading without being too vanilla. It's definitely worth a poke around. The only bit I found that was lacking in content was the FAQ section (in particular the last question :-).

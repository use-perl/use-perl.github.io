New commercial product, and some bookmarks

gnat on 2001-04-20T14:19:20

MySQL Data Manager is a generic web front-end for MySQL databases. More interesting than that, it's a commercial product written in Perl and using timbot (Bunce)'s excellent DBI module. It's not often I see new commercial Perl software, so I'm glad it's still being produced.



Here are two web sites I've recently enjoyed (warning: minimal Perl content ahead :-):

  • lileks.com. Hilarious rips on images from the US of the 50s and 60s.

  • half.com. Lots of cheap music, books, and DVDs. I've been on an Irish music bender recently, and I've found a lot of stuff I was after for $3 to $8.

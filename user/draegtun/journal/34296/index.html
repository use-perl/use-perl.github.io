mirror.ac.uk has closed down

grinder on 2007-09-01T09:11:39

mirror.ac.uk was shut down in July as part of a cost reduction drive. The press release is here:

http://www.jisc.ac.uk/news/stories/2007/03/news_mirror.aspx>

All current users of this mirror must migrate to another mirror. We've taken it out of the CPAN mirror list, but there's nothing that can be done about existing configs: these have to be edited manually by each user.

CPAN mirrors

Aristotle on 2007-09-01T14:03:21

Check mirrors.cpan.org for an up-to-date list.

My favourite approach is to use CPAN::Mini on one of my machines to keep a local minimised mirror. I just update it infrequently, and whenever I want to install a module, the “download” is infinitely fast. Very convenient. And the disk space it takes is peanuts nowadays (about 720MB right now).

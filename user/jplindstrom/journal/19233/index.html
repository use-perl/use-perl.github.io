CPAN Reviews: Write one today!

jplindstrom on 2004-06-13T12:56:44

What's the easiest way to contribute to CPAN?

Make it easier for people to find the good stuff; write a review of a module you use.

If you search CPAN, find a module and end up using it, make it a habit to write a review! You probably still have the browser window open and it's the least you can do to say thanks for the work you didn't have to do.

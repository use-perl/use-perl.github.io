Venetian Blinds

jplindstrom on 2004-11-04T12:58:24

The latest for-fun project is Venetian Blinds, which keeps track of blind levels for us when playing Texas Hold'em. Because, eh... that's such a chore :)

Screen shot

It took a couple of hours to write the first version, and another hour or two to fiddle with the speech API and Jouke's Win32::SAPI5 (which could use a few more examples in the synopsis btw). Finally I decided against using it because of the messiness with external dependencies and huge downloads to get a decent voice.

I'm also in the process of writing a Win32::GUI and The GUI Loft tutorial with this program as a case study. It's not nearly finished, but as extremely incomplete drafts go, it's not that bad.

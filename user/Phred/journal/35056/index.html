Leopard and Catalyst

Phred on 2007-12-07T05:16:59

I've upgraded my laptop to Leopard, and almost everything just worked. Only PostgreSQL broke, since Apple had the foresight to remove the /etc/sysctl.conf file (huh??). Good job on Leopard except for that little oversight. Looking forward to learning dtrace, I've pushed dproffpp about as far as it will go and need a better measurements tool. Spaces is cool but not as good as linux virtual desktops (the number of keys that I can map is restricted)

The catalyst work is going better. Getting everything setup and all plugins, etc playing together nicely was painful. Once it is up and running though, new work goes much faster and is relatively pain free. Sorry to you catalyst folks who had to read my rant a few weeks ago :)

Update - ssh also broke, and it doesn't seem quite as snappy as 10.4.

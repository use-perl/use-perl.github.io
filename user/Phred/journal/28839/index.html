Perlnow

Phred on 2006-03-01T22:57:45

Last night at the San Francisco Perl Monger's meeting, Joseph Brenner gave a demonstration of Perlnow, a set of Emacs extensions he developed to speed Perl development. I installed it today, and I'm already having fun with the various macros such as 'perlnow-run-check' (perl -wc yourfile). I've started using Emacs seriously in the last few weeks after a 10 year hiatus. It took a short while to get back up to speed but I have to say it's improved my productivity being able to integrate development tools such as Perlnow.

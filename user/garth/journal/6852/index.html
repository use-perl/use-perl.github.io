JournalX v0.2

garth on 2002-08-02T03:26:00

I've uploaded a new version of JournalX, with a few new features, and a whole heap of cleaning up of various bits and pieces. The new version, along with the source, and a few screenshots can be found here.

The main changes are:

- Fixed main window, to allow resizing - Added ability to delete existing entries - Added ability to edit and overwrite existing entries - Updated offline entry maintenance, now able to selectively upload or delete entries - Comments link can now be clicked - Changed menu items

The instructions for use are the same as the last version, which are here.

Editor for Mac Perl

jlttb on 2001-10-23T13:56:27

Everything's easy when you know how!

Been using MacPerl on my home and work Macs for a couple months now, and had set BBEdit as the text editor. Work machine got upgraded (to sys 9, but that's progress) and the BBEdit hook was gone. By which point I'd forgotten how I originally set the editor. The documentation refers to setting helper apps in Internet Config, but at least for me, that wasn't enough!

After a fruitless search of the documentation and web yesterday, I went home and copied my home Mac settings, and was able to replicate them this morning at work!

For the Record Go to Internet Config (in Sys 8) or the Internet Control Panel (in Sys 9) and add a new helper app - which must be called 'editor' (that's the hard part, don't say text or .txt or .pl), then use the navigation buttons to link in your editor of choice. Then it works great!

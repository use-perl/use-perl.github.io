SAP Shop

ajt on 2005-03-14T11:16:16

Today I tried to order some SAP books from the SAP Shop. The shop is a joke, and I've no idea if anything worked. I don't mess about with sites when it's my credit card that's on the page. The SAP web site is normally a joke at the best times, but their "shop" is really awful, you'd think they'd never done a user interface - then you realise why the normal SAP interface is so awful - they don't have a good design to start from...

You'd think their training centre could sell you books, but apparently that's too much for them. I don't really want the books in particular, but work is paying for them, so I'm not going to refuse. Normally I don't mind buying the odd book from Amazon and then claiming my money back, but buying from SAP is proving more awkward.

XML::RSS::Tools next beta

ajt on 2003-09-28T17:17:38

I uploaded another test version of XML::RSS::Tools to CPAN today, this version incorporates HTTP Proxy support, and some other minor changes. For the final release of this build I also want to make it work with Module::Build as well as the traditional make approach.

I'll do some more minor changes, and try and work out a way of testing the proxy support (automatically and manually), and then I think it will do as a release.

Just Doing It

ajt on 2005-12-03T23:41:21

At the recent London Perl Workshop there was an entertaing talk, with some interesting modules in it. I use the train to get to and from work and using the official web sites to get details in painful. There is the much better accessible web site but that's still requires human input to get details. In Greg's "JFDI" talk he mentioned his WWW::LiveDepartureBoards module which allows you get train departure data via screen scrape into machine usable format.

Most useful, I've already knocked up a short script, all I need to do know is to get it to email my mobile when the trains are running late and I can spend a few extra minutes in bed in the mornings too!

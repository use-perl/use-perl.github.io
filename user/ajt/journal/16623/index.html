Browsers are dead, long live MSN...

ajt on 2004-01-04T14:04:54

In a typically poor piece of journalism some analysts is claiming that web users are now surfing using software other than a browser. Given that the Internet predates the "Web", and that there is and has always been a great deal of other traffic over the Internet, I find these figures and claims quite silly.

While there is no doubt a lot of Internet traffic generated by tools such as MSN, AOL IM, ICQ, Real and so on, however I think that email, spam and virus activity probably dwarfs the total activity of these so called "new" kings of the web.

Another point

ajt on 2003-01-11T13:44:52

My XML:RSS::Tools module made it up to CPAN as another minor-point upgrade. I'm pretty happy with this version, it's starting to get a complete set of documentation, some proper examples, and the code seems stable enough, no one has complained yet.....

Spurred on by Bob's article on XML.com, I've been able to create a single XSLT stylesheet that can handle all 7 versions of RSS, for generating simple XHTML. I had to add an option in my module to allow the user to disable RSS normalisation to make this functional, and I took the opportunity to extend the test section while I was tinkering with the version code.

My next task is to install the latest build of XML::RSS, at the moment all I get is failed dependencies on both my Win32 and Linux boxen, but I will overcome the problem - it's only a matter of time.

Other than doing some editing work on an old site of mine, I think today will be a day to watch more of my Red Dwarf (season 1) DVD!

Sprog 0.10

grantm on 2005-06-15T00:42:27

Sprog 0.10 has just been released. The main crazy new feature is the '--nogui' option which allows you to run your machines without a GUI (eg: from cron, .profile etc). Full release description and download links here.

With Gavin Brown's help, I'm finding out how to create RPMs on my Debian system for use by Fedora Core, Mandriva, SUSE etc users. If you want to try one out and let me know if it works for you then you'll find the latest build here.

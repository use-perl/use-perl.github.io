Lightning Talks

grantm on 2005-03-14T23:38:47

Wellington Perl Mongers had a lightning talk evening last night. We're not a huge group so having 9 people sign up to speak was pretty good. Unfortunately there were three no-shows on the night (some through illness, some unexplained) so it was all over pretty quickly. However it seemed to go pretty well and there was plenty of time left at the end for socialising - which doesn't usually happen. So all things considered, I'd rate the evening a success.

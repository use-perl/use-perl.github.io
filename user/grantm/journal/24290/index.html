New XML::Filter::Sort

grantm on 2005-04-20T09:19:04

I got a report today that my XML::Filter::Sort module had a reasonably serious bug which caused it to corrupt elements with the text content '0'. Despite not having looked at the code in nearly three years I had a test case added, the offending line identified and a patch in the hands of the original user within the hour (nyah nyah ESR).

The new code is up on CPAN now but sadly I expect it will get the thumbs down from cpan-testers on account of the current state of XML::SAX::ExpatXS :-(

Of course the fact that the bug has lurked for so long suggests that precious few people are using the module.

Update: Turns out another very similar bug lurked in another part of the module, so another release was needed first thing this morning :-( And here was me thinking I wouldn't have to look at it for another three years.

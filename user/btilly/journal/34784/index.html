spod5 (Re: A/B Testing)

Corion on 2007-10-30T07:29:13

I really like spod5 by Philipe Bruhat to convert POD to S5. Placing images is a bit unwieldly, but if you want exact image placement, you can just include the raw HTML to display an image at a given position.

The fun thing with S5 is that you can get automatic (bad) translations for talks using any of the web translation services, as the whole slideshow is just one HTML page.

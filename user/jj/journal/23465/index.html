Update to perldoc.perldrunks.org

jj on 2005-03-02T22:09:28

Following user feedback I've made a few updates to the http://perldoc.perldrunks.org documentation site, the main changes are:

  • Improved interlinking of 'perl' manpage (thanks Aristotle [1])
  • Updated syntax highlighting rules to create links for Perl function names and core modules
  • Moved 'perlop' page to new top-level section 'Operators' (thanks Bart [2])
  • Moved 'perlrun' page to 'Overview' section
  • Moved 'perltodo' page to 'History/Changes' section
  • Fixed numerous broken links in functions pages (thanks Aristotle [1])
  • Added some missing functions ('getpwnam' etc)

Please post any bug reports / comments here or email jj@jonallen.info

Update to Perldoc search

jj on 2005-07-12T23:15:02

I've made a change to the search engine on perldoc.perl.org.

If you enter a Perl function, manpage, or core module name intp the search box the site will now bypass the results list and go directly to the relevant page, a much quicker form of direct navigation than using the menus.

As an example, a search for pod will initially bring up the perlpod manpage with a link to access the full list of results.

Norton Antivirus message caused by moving shortcuts

lachoy on 2005-01-20T11:38:26

Norton AntiVirus 2005 does not support the repair feature. Please uninstall and reinstall. - When is a shortcut more than a shortcut?

This message can happen if you have changed the default location for the Norton AntiVirus shortcuts in the Windows Start Menu....
If you have moved the shortcuts from the default location, then move them back to their original location to fix the problem.

Who knew shortcuts had so much power?

Last day at ZSI

domm on 2006-02-01T06:47:15

Yesterday was my last day at ZSI. I worked there for aprox. 8 years. When I started, I wrote some scary cgi-scripts using a paramater parser I got from Matts Script Archive. Now I write mod_perl apps using OO, tests and docs, and half of CPAN. I learned quite a lot there (all by myself (and from O'Reilly, perlmonks and YAPCs), as I was the only coder). It definitly was a nice job, with funny and intellegent co-workers (all social-science types).

In half an hour I'll leave to go to my first day at my new job. I'm looking forward to it, not only because the pay is better, but because I'll be working together with a lot of other coders (some Perl, some Java (which I won't have to touch, thankfully)).

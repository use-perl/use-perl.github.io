Printers for TPR

brian_d_foy on 2004-07-09T05:05:36

I spent the day talking to printers, and I think I know who is going to print the first issue of The Perl Review. Most shops quoted the same price, but it came down to this question:

Me: So what happens if I blow it and give you the file late?

Them: We have to do it faster.

Me: How much extra does that cost?

Them: Um... [pause] ... nothing. We just do it faster.


Well, okay then. :)

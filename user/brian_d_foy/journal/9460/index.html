SSH for the Palm

brian_d_foy on 2002-12-14T17:30:15

My Handspring is even more useful since I have found Top Gun SSH. It works. I actually get a terminal right on my PDA's screen.

It has its own terminal type, and PINE does not like it, but other things seem to not mind. The first command I ran was a simple hello world.

perl -le 'print "Hello World"'


The terminal font is a bit wierd, but I will get use to it. If I want to do Perl, though, I need to learn a lot more Graffiti to get all of those special characters. Maybe I should switch to Ruby---I know how to make full stops.

The ssh client cannot do RSA keys and still have acceptable performance (which is pretty darn speedy anyway). When will these PDAs get the nifty 1 GHz chips? So what if they could fry eggs---with a griddle expansion module that is just another feature.

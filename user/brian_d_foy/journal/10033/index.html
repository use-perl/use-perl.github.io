Tk regular expression explorer

brian_d_foy on 2003-01-17T05:36:58

A long time ago, John Klassa, main monger of Raleigh.pm, wrote a Tk application (twiddle-regex) that highlights a string based on a regular expression, shows what is in the match variables, and so on. Very cool.

Now that I have X11 on my Mac laptop, I get to show this off to my Learning Perl courses.

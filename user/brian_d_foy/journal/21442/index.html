LiveJournal for The Perl Review

brian_d_foy on 2004-10-21T04:21:58

After wasting a lot of time setting up, configuring, testing, reconfiguring, and re-testing various blogging set-ups for The Perl Review, I capitulated. I'm not in the blogging business, and I don't want to manage software and servers.

I'm going to let LiveJournal do it for me. I set up an account for perl_review, as well as a the_perl_review community (which you have to join, but that's no big deal). LiveJournal handles all the posts, RSS feeds, and all that good stuff.

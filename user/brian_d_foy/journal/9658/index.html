New PAUSE feature

brian_d_foy on 2002-12-29T12:48:25

A long time ago I wrote that I was playing with the PAUSE code. My first change is now active---very exciting for me. I like pointing at things and saying "I did that", and it is even better if it is true.

PAUSE admins take care of the mundane task of adding users to CPAN. I made that a little easier by making it possible to do that with no cut-n-paste. The process works much like submitting a namespace request---the requestor fills out a form which mails a message to the PAUSE admins. Since the script mails the admins instead of the requestor, the script can collect all the right data (or fail), format everything in the right way, and construct the link we need to approve the addition. Very nifty indeed.

In truth, I think I did about a third of it and Andreas filled in the rest based on how he wanted things to look. I; however, now feel much more comfortable with the PAUSE code. It does the job pretty well, so I do not see me making any big changes, but there are a few things on the TO DO list that might need a bit of maintenance.

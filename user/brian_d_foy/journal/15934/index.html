Perl Code and the Cosmic Radiation Background

brian_d_foy on 2003-11-22T04:38:19

I fooled around with my text density program more last night. I turned the image on it's side so the lines of the text stretch across rather than down, and then I added color (instead of grayscale)---blue for the areas of massive whitespace to red for nasty clumps of characters.

With the false color, the images look like the false color pictures of the beginning of the universe. Very cool.

I wish I could track the text character by character so I could make movies that looked like germs (or whatever) growing in a petri dish.

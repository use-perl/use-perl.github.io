Pseudo-new Mac::iTunes

brian_d_foy on 2005-07-15T07:19:28

I've released a new Mac::iTunes, but don't get excited because it still sucks. I added some bits to check for the library version before it tries to parse it and to die if it doesn't understand that version of the library. Along with that I noted in the docs that the iTunes world has moved along and I haven't kept up.

The problem is still the new binary format that Apple choose and which has been completely inscrutable to me. If anyone wants to work on it I'd gladly acccept patches.

If anyone wants to write the Mac::Glue counterpart to the slow and yucky Mac::iTunes::Applescript, I'd appreciate that too.

So, don't download it. Don't use it. Keep in mind it's merely version 0.85 and that's a long way from 1. :)

EPrints

inkdroid on 2003-01-20T20:24:53

Eprints is an application for building an eprints archive ...it's written in Perl, and available for use under the GNU License. An Eprint Archive is a collection of digital documents and metadata whose contentst are interoperable with other eprints archives. Metadata can then be harvested into global "virtual" archives that are seamlessly navigable by any user (just as a commercial index or abstract database is navigable, but with full-text access.

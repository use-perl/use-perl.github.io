Kinda-new Mac::iTunes makes a smaller CPAN

brian_d_foy on 2003-01-03T20:59:24

If you already have Mac::iTunes, you do not need the latest one. I changed the test mp3 to a much shorter version of itself now that I know how to edit mp3 files.

Once PAUSE deletes the previous version of Mac::iTunes, the Schwartz Factor should take a nice jump. Anyone else who has large distributions on CPAN should consider removing earlier versions since the BackPAN archives them.

Maybe most of this will not matter when we all have DVD-Burners---CPAN only takes up about a thirrd of a data DVD.

Palm's Mac support sucks

brian_d_foy on 2002-12-11T23:39:09

A couple of weeks ago I ordered a Xircom SpringPort Wireless Ethernet module. I was not sure the price was right because it was so low, so I thought it might not show up.

When I got back from my trip yesterday it was waiting for me. I needed to charge it for two hours before I used it so I went to bed while I left it in my HandSpring cradle.

Today I tried to use it. Oops! I use a Mac---no network HotSync for me. The Apple iSync stuff apparently is just a front end for the HotSync Manager. The HTML browser that comes with it requires a proxy server that I have to sign up to use---no thanks. I need to find some network tools for this thing. Maybe I could reverse engineer the network HotSync functionality so I can support it on the Mac.

So, when is Perl going to be on the Palm OS?

BeOS, Ruby woes

djberg96 on 2003-01-14T21:07:22

The good news is that Ruby 1.8 builds on BeOS 5 without a hitch. The bad news is that, despite several tips from a couple kind folks on the mailing list, I cannot get custom extensions to load.

Oh, they build just fine. No warnings, no errors. I've tried everything that I know - statically linking the code into the ruby interpreter, slapping in -lbe and -lroot, messing with the $LOAD_PATH, sacrificing coca-cola to the computer gods - none of it worked.

If I *do* ever get it to work, Linux is going to be very lonely for a while.

How to fix your mangled MakeMaker install

schwern on 2003-04-11T01:20:56

Grab the 5.5.3 sources and

        cp -r /path/to/5.5.3/src/lib/ExtUtils/* /path/to/5.5.3/install/lib/ExtUtils/

That should fix ya.

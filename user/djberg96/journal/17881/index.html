blastwave.org

djberg96 on 2004-03-12T17:37:02

A short while back, while bemoaning my 64-bit blues on comp.unix.solaris, someone mentioned blastwave.org. It's a repository (well, it holds links to mirrors and other stuff) for prebuilt Solaris packages.

How is this different that sunfreeware.com? The difference is that the blastwave packages are built properly with regards to the runtime library paths, while the sunfreeware packages generally are not. Also, there tend to be more packages available from blastwave.

Ah, I can feel a tranquil feeling spreading over me as I write this. OHMMMMMMMM, OHMMMMM....

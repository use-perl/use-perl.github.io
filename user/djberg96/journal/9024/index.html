More CPU please

djberg96 on 2002-11-19T18:35:14

During my sick day yesterday, I spent some time playing Battlefield 1942 online. I quickly realized one thing - a 400 MHz P2 just ain't gonna cut it. Any more than 12 players and things get way too choppy too fast.

I could install Windows 2000 to get the power of the second CPU going, which would help, but I'm not sure how much. Frankly, I think even some of the latest and greatest chipsets would groan under the strain of this game.

Maybe next year I'll upgrade my motherboard for a dual Itanium setup. Yeah, that's it. That's the ticket!

find without the popup

prakash on 2004-10-07T21:10:57

I've always disliked the find popup box resulting from pressing CTRL+F. That was until Mozilla/Firefox had added the feature where you can search just by typing a '/' followed by the text you want to search (just like in vi). To search for links you don't even have to type the '/' character. Just start typing the text.

I haven't had to use the find popup since several months.

/prakash

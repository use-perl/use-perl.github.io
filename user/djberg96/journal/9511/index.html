Mozilla

gizmo_mathboy on 2002-12-18T03:19:17

I would recommend Mozilla over Netscape.

For one thing it's easier to stop pop-ups in Mozilla (a check box in Preferences vs. a manually added line in users.js, which has to be created by the user). For another, it has much more useful features and fun features. I'm running 1.3 alpha and it seems fine. Sometimes I get crazy and try the nightlies.

If you're just using Netscape 7 for a browser then get Phoenix. It's Mozilla just stripped down to the browser stuff.

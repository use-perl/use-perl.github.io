DSL issues

djberg96 on 2003-01-31T14:50:10

Ever since I got Qwest DSL installed, Gamespy has warned me of the fact that my external IP address does not match my internal IP address. Being somewhat ignorant of How Things Work, I figured that was just a normal state for DSL, or perhaps it was some wonky configuration being used by Qwest and/or MSN.

Well, for the first time last night I got warnings from sourceforge about this issue and someone on IRC told me I needed to get it fixed ASAP.

Oops.

Update: A co-worker tells me that this is, indeed, normal because I have an external dsl modem, which acts more like a dhcp server for all the computers I might have hooked up.

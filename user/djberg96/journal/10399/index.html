anjuta

djberg96 on 2003-02-05T03:37:30

Always on the lookout for the ultimate editor, I decided to give Anjuta a try. I have to admit the interface looks very slick, and the code folding is very nice.

It's primarily aimed at C/C++ programmers, but there's syntax highlighting, code folding and some method tagging support for Perl, Python, Ruby, etc. Now just add better CVS support, a debugger and configurable syntax highlighting and I can quit using vim forever. :)

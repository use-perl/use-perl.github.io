Data-Phrasebook-0.23

barbie on 2005-08-09T22:53:53

The latest release of Data::Phrasebook should be hitting CPAN now.

The update was for a bug/feature that I thought I'd already corrected, and also to add the support for listing the currently loaded keywords, or the keywords in a specified dictionary.

The loader plugins for INI and XML files have also been updated. Both with the keywords support and documentation for how to use continuation lines.

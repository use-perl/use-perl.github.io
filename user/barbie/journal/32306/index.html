LUGRadio Live 2007

barbie on 2007-02-02T17:15:01

For those that might be interested, one of the best Linux conferences, LUGRadio Live, is scheduled to be held in Wolverhampton 7th/8th July. No venue details as yet, but there is the possibility they will be moving on from the university. If all goes to plan, this will be the weekend before GUADEC starts, so July is going to be quite an eventful month.

If you want to submit a talk, send applications to show-at-lugradio-dot-org.

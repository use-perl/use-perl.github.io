Spango The Wonder Badger

barbie on 2004-01-15T10:18:17

When at tw2, we had a bit of a joke reference, which involved a badger called Spango. I can't remember where the story came from, but it stuck as an in-joke for ages. When tw2 ended, the core funsters who had entertained on our internal mailing list, foobar, set up the Spango mailing list via Yahoo. Why mention this, well my review copy of the Badger Book arrived in the post. I just can't help thinking the badger on the front is called Spango :)

Having used Template Toolkit for quite sometime now, it'll be good to start learning how to use it properly. The docs online and bundled with TT, have got me sorted in virtually every instance I've needed them, so I'm intrigued to see where this book takes me. I'm sure they'll be some overlap, but it's the bits I didn't know about that I'm looking forward to. I hopefully have a couple of quieter weeks coming up so plan to get a lot of reading done.

The Dangers of Upgrading

barbie on 2006-03-02T12:35:18

I'm not having a fun time this morning. After getting the persistent request to upgrade with Bundle::CPAN, I decided to do just that. The install failed, but rather than be left in a useable state, it now means I have a crippled CPAN.pm. I then opted to try PPM and discovered that now has an 'rvalidate' error. Thankfully PPM3 seem to be working, but it doesn't have access to everything i want. CPANPLUS won't install because it needs interaction, which is now broken thanks to the borked install of Bundle::CPAN.

Until I can figure out the culprit of the bad install, I'm going to have to download everything I want by hand. Not fun :(

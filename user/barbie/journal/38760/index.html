CPAN Testers now on Github

barbie on 2009-04-05T15:30:57

On Saturday I upgraded git-core on the cpantesters.org server. As a consequence none of the repos on the server now work :( Thankfully I have my local copy, and a backup copy, so nothing has been lost. To make life easier for myself, in case it happens again, I've now ported all the CPAN Testers distros that I look after onto Github. To see them all (together with a few extra repos) check my profile page.

In the coming weeks I'll also be storing all my other CPAN distros on Github, so you'll then have another way to send me patches :)

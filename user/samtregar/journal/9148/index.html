Dead Machine

samtregar on 2002-11-26T05:40:47

It's sad when an old machine finally bites the dust. Maybe this is a sign of mental illness, but it makes me pretty unhappy that the PII/400 my wife was using is now dead.

I don't know what's wrong with it, so I'm not sure what I can salvage. The hard drive light comes on and stays on but the power light never does. Once on, the machine won't turn off via the power button and has to be unplugged. I never even see a BIOS screen. I've tried juggling the memory around, reseating the processor, removing all the cards and then reinserting them one by one, starting with the video card. Nothing.

I suppose it might be the power supply. Maybe I'll try to switching in a known working one to see if that gets it going.

Hum. I guess I'm in the bargaining stage. I suppose later I'll be cursing it for leaving me...

-sam

XML::Validator::Schema 1.06 In The Can

samtregar on 2004-04-21T22:13:33

I finally got a chance to go through all the bug reports and patches accumulating in RT for XML::Validator::Schema. The result is XML::Validator::Schema v1.06. Aside from bug fixes, this release also adds support for positiveInteger, negativeInteger, nonPositiveInteger, and nonNegativeInteger.

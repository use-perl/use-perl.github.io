HTML::PopupTreeSelect 1.4, the Chris Nandor edition

samtregar on 2004-04-30T19:21:56

HTML::PopupTreeSelect 1.4 is out. This release contains a bunch of cool stuff from Chris Nandor (better support for sub-classing, multiple roots and a new variable for the template). I added an 'inactive' flag for the node structure to support displaying nodes that can't be selected and kicked it out the door.

I also updated the demo page, so if you want to know what all the fuss is about look here: http://sam.tregar.com/hpts_demo.cgi.

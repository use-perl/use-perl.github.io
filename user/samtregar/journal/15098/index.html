XML::Validator::Schema 1.04 is here to stay

samtregar on 2003-10-06T23:03:22

XML::Validator::Schema 1.04 is up on CPAN. I've added support for a few more schema constructs and fixed a bug in the cache implementation.

I think once I get line numbers into the error messages (incomplete SAX parser implementations, grumble, grumble) I'll be statisfied. After that I expect I'll only work on the module in response to user requests, be they bug reports or patches. So this is my notice to you, fair reader, if you want a better XML Schema validator then you better be prepared to work for it!

-sam

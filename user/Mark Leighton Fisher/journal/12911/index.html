Slashcode Journal RSS Feeds

Mark Leighton Fisher on 2003-06-17T23:05:27

The secret sauce for extracting RSS feeds from a Slashcode-based Journal like mine is to specify the user ID and content type to journal.pl, as in
     http://use.perl.org/journal.pl?uid=4252&content_type=rss
where (in this example) my UID is "4252" and the content type (actually "content_type") is "rss".

There may be another, more elegant method — but I haven't found it yet.

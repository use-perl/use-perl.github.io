Microsoft Web Browser Control Creates a New MSIE Instance

Mark Leighton Fisher on 2006-11-10T17:23:15

When you click on a link with target='_blank' displayed inside of a Microsoft Web Browser Control (shdocvw.dll), a new instance of Internet Explorer is started to display this link. This Internet Explorer ends up as a child of the svchost.exe DCOM Server Process Launcher. This means that Internet Explorer does not go away when the program hosting the Microsoft Web Browser Control goes away.

Is this a security risk?

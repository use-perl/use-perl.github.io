Nvu: HTML Tags-On Editor

Mark Leighton Fisher on 2006-07-14T16:50:16

Nvu is an Open Source HTML editor and Web Authoring System for Windows, OSX, and Linux. Nvu brings back a feature that I had missed dearly from HoTMetaL – "tags-on" view. This is an editable mode displaying your page's final layout while simultaneously showing the tags used to create that layout. I had used tags-on mode all the time in SoftQuad's HoTMetaL, but SoftQuad apparently buried HoTMetaL when XML first became prominent. (Now SoftQuad is part of JustSystems...)

Now I have an HTML editor with tags-on view which is more customizable than HTML-Kit (my previous HTML editor of choice). Thanks and kudos to the Nvu developers – Nvu looks like genuinely useful software to me.

Acme and Root-tail

cog on 2004-07-26T10:37:01

Today I'd like to tell you about two wonderful pieces of software I can just wish to have known about before :-) (you may have heard of them, of course)

Allow me to say that these were the easiest installations I have ever done :-)

ACME

No, not that Acme, this acme :-)

With this little wonder, I am now finally able to use some of the buttons in my laptop that were designed for windows: volume up, volume down, lock screen, etc.

Unfortunately, I can still only use four of the eight buttons it has :-| Don't know what's wrong with the others...

Check it out :-) It's worth the time :-)

Just remember to use the acme-properties to configure it and the acme command to activate it :-)

ROOT-TAIL

And with this little fellow, my background now tails some files (see screenshot).

Do know that "some desktop environments open a virtual root window" blah blah blah :-) Basically, just go to Nautilus (if you're using it) and disable the "show desktop" feature :-) Otherwise, Nautilus won't let anything else draw in the background... "It's MY background!! Keep away from it!" :-)

You can choose fonts, colors and other stuff for each file :-)

This is all very, very cool :-)

Thanks to Alberto for letting me know about this :-)

Regards to you all :-)

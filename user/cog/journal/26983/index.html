Some more feedback on my previous iBook

cog on 2005-10-03T10:48:34

So I used to complain a lot about my previous iBook, right?

Well now I have a new one, and all those problems disappeared...

Meanwhile, right before that old iBook was sent back for repairs, another iBook's hard drive failed, so the guy used the old one to store all his data and work with it while his would be repaired.

I keep hearing him say "I have absolutely no idea how you put up with this shit for so long."

Turns out, it really wasn't me, it was the machine :-)

BTW, I happen to know why I put up with it for so long... I simply didn't know whether all Macs were like that or whether it was just that one.

Not that mine is perfect, though...

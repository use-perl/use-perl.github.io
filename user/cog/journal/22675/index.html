Devel::Cover

cog on 2005-01-12T10:56:17

I was asked about tests (which I do on a regular basis) and then I was asked about Devel::Cover.

The truth is that I'd heard of it, I had an idea of what it would do, but I had never used it, so I gave it a try.

It's cool, because you can actually see which parts of the code are or are not being tested. Cool! :-)

To my astonishment, it actually found some lines of code I thought were being tested but were not :-)

Any tips as to what you do with Devel::Cover will be mighty appreciated :-) I'm curious on how people use it (because it seems there's more than one way to do it...)

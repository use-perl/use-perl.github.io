Perlplexmenot

cog on 2005-06-15T14:06:04

davorg points out the Perlplexer, a Perl quiz that, according to whomever created that page, will "hunt for the 5 best Perl / Open Source programmers"

If you disregard the fact that you'll be taking a Perl quiz in a PHP site, the "Welcome to the ultimate Perl testing machine!" message sounds interesting...

As you start seeing the questions and their relative possible answers, your jaw lowers a bit...

"WTF", you say.

Some of the questions are broken.

Having 19 or 20 correct answers make all the difference between a "We think you should go back to 'Perl for DUMMIES'" and a "Ever thought of writing a Perl book?" result.

I never thought it would boil down to that :-)

Uri had a great idea, on irc:

<uri> i am getting that test. name: i hear your test blows. email: badperl@sysarch.com
<uri> so i will insult them before they can insult me :)

If you want to laugh a bit, take the quiz.

Follow Uri's suggestion if you want O:-)

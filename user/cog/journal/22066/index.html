How the world changes...

cog on 2004-11-30T12:03:41

In my first year at the University, I remember I would only use Windows, and all I'd do with it was playing games... There was a friend of mine who was a great hacker (nagra), who would only use Linux and command-line interfaces, practically...

Now, years later, I was having an argument with him... He's using Windows, I'm using Linux. He uses all-fluffy tools... For reading mail, he's using becky; I'm using mutt (with procmail, postfix and fecthmail); for browsing, he's using netcaptor, while I'm using Firefox with a couple of extensions.

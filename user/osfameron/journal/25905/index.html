Pod2IT

osfameron on 2005-07-27T15:32:28

Complimenti to the Italian perlisti on the release of the pod2it project onto CPAN. This release is a first step to translating the most important Perl PODs, and is an important strand of the Italian Perl advocacy plan.

There's some more information here which includes a pointer to a nice little patch to Pod::Perldoc allowing you to specify a language flag to perldoc.

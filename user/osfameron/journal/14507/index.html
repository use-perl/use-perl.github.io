CV in Moveable Type?

osfameron on 2003-09-05T12:42:43

Thought:

I've wanted to automate my CV for a long time and it just occurs to me that it could be done in MoveableType as a blog. Each job could be a blog entry (in category 'job'), same for training, education, skills etc. Then just stitch together a sufficiently clever template and voila', instant CV.

With appropriate metadata and some custom plugins, I could set up different templates to filter e.g. for skills:

  • Only the technical skills
  • The full entry for only the most important technical skills. Only the excerpt for all other technical skills and languages
e.g. to repurpose the CV for different classes of employer.

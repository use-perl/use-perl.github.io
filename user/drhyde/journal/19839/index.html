Mozilla bugs

drhyde on 2004-07-14T09:00:48

Recently several people have been going on about how great it is that some of the sheople are ditching IE and switching to Firefox or some other Mozilla variant. And they're right, it is great. However, Mozilla isn't a panacea. There have been several posts to bugtraq and fulldisclosure this week about vulnerabilities in Mozilla. Some those bugs only affect Moz when running on Windows and also affect IE, so you have to question whether the browser is at fault or the OS. Trouble is, getting users to switch OS is going to be a lot harder than simply using a different pointy-clicky-drooly browser.

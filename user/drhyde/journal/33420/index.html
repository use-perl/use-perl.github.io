New Bryar features

drhyde on 2007-06-04T13:48:42

I've been hacking on Bryar some more. The version now running on my own site supports user-defined keywords (which some people call 'tags' for no apparent reason) and also has a rather crude heuristic to let you figger out whether the user is on a mobile device and so might need different templates.

More details in my journal which is where I would prefer to see any comments, although you can leave them here too. Provided I don't find any errors I'll release the new stuff to the CPAN soon.

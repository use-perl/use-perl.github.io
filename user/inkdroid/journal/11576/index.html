chicago.pm tech meeting

inkdroid on 2003-04-10T16:03:02

At the next chicago.pm meeting (04/21) I will present a short tech session on SOAP. If you come you can expect to learn what SOAP is, why it might be useful to you, and how to create SOAP clients and servers using Paul Kulchenko's lovely SOAP::Lite. If you'd like to know more about SOAP before deciding to come take a look at the intro in the spec, and the SOAP::Lite docs.

More importantly, Marsee at O'Reilly has donated some books to chicago.pm which are going to be given new homes.

  • Programming Web Services With Perl
  • Embedding Perl in HTML with Mason
  • Linux Server Hacks
  • Perl Pocket Reference
  • Perl CD Bookshelf, V3.0

The talk won't be longer than an hour, during which you can eat & drink. I'll be using handouts and a whiteboard unless someone can get a hold of a projector we can use. The meeting will be held at Coogan's Riverside Saloon, 180 Wacker Drive, two blocks up from the Civic Opera House. Not sure about the time, but there will be another announcement closer to the date. More info available at chicago.pm.org

MakeMaker release imminent. Try it with your module.

schwern on 2005-03-15T05:26:03

ExtUtils::MakeMaker 6.25_11 was released today and its looking ok. It might make it as 6.26. You should install it and try it out with your module, make sure whatever horribly hackery you're doing still works.

Its been almost a year and a half since the copy of MakeMaker shipped with Perl got updated. This is the first release since 6.21 that I feel is ready to be put into 5.8.x. Best knock it around now.

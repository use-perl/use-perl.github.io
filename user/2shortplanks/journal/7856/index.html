Acme::Test::Buffy

2shortplanks on 2002-09-20T12:28:47

Okay, so in order to annoy acme, who's currently trying to list all the Acme modules for his talk at YAPC::Europe, a few of us having been trying to, um, pad the list. And this module is my attempt.

As I think all Acme:: modules should be useful (if not actually do something useful themselves,) I've made this one as a simple example of how to use Test::Builder and Test::Builder::Tester to create testing modules. This means that this module contains lots and lots of comments - about three times as many comments as code. And I bet those contain many typos and bad explanations.

So, even though this is a "joke" module, I'd be pleased if people would consider submitting bug reports and patches to this...even if they are just patching a typo in the comments.

Right, better get ready for my talk.

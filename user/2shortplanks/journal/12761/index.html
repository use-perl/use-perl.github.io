Reason to PAUSE for thought

2shortplanks on 2003-06-11T22:52:38

Just released an updated version of Template::Plugin::XML::LibXML onto the CPAN. A few bug fixes and documentation changes, nothing major.

PAUSE scared the hell out of me by emailing me a list of errors saying it couldn't index a load of stuff as I didn't have permission to claim those modules. It took me a few seconds to realise that it was merely stating that it wouldn't index things like XML::LibXML::Node - modules that are part of XML::LibXML. PAUSE was getting confused as the plugin includes 'package XML::LibXML::Node' declarations to directly place methods in those module's namespaces. I guess you get the same deal if you bundle things like Test::More with your code.

Haven't got slides done, and haven't packed yet. Bother.

PL/SQL idiom

jdavidb on 2003-09-17T19:10:43

In Perl:

foreach my $word (qw(this that whatever))
{
  dosomething($word);
}

In PL/SQL:

<>
DECLARE
  TYPE wordlist_t IS VARRAY(somenumber) OF
                     VARCHAR2(#);
  wordlist wordlist_t
    := wordlist_t('this', 'that', 'whatever');
BEGIN
  FOR i IN wordlist.FIRST .. wordlist.LAST
  LOOP
    dosomething(wordlist(i));
  END LOOP;
END foreach;

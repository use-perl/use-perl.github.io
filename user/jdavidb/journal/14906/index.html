perl 5.8.1 released

jdavidb on 2003-09-25T15:36:11

Jarkko has released perl 5.8.1, and it has propagated to CPAN. Thanks for all the hard work!

Changes include randomizing hash orders for security (they were never supposed to be reproduceable, anyway), module updates, and Unicode updates. Most important to me, in my present environment, is that 5.8.1 compiles cleanly for Cygwin out of the box!

Small steps

runrig on 2005-03-04T07:03:09

New gig is going okay. I'm keeping busy doing among other things maintaining the perl I wrote many years ago. There's been a bit of code decay since I've been gone, with some of the scripts having the 'use strict' commented out (whenever it 'got in the way') and uninitialized warnings popping up (I've corrected those things on the scripts I've worked on so far). Heavy refactoring, however, is discouraged, since the long term plan (so far) is to convert them to java (and I can't wait to see how many man-months that will take). I can't complain too much, since I'm the only perl expert there, though the couple of guys there who have been maintaining these scripts (and learned perl from them) agree that perl is the perfect (appropriate, anyway) tool for this task, and I've even been helping with unrelated data-munging perl that one of them has been writing.

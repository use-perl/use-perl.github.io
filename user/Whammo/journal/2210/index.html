We've added a new player to my coed volleyball team, so I decided I was going to redo the game plan with animated GIFs, rather than redistribute the old ASCII drawings I normally do. (My scheme is actually a little simpler than the standard coed lineup, but because it's not the standard, it takes some getting used to.) The benefits are a lot more noticeable when you can see the plan in transition, rather than some before and after shots.



I wasn't looking forward to a lot of hand-poking in GIMP, though. That's when I remembered I had Wallace's book on Programming Web Graphics. A quick install of ImageMagick and PerlMagick from my cooker mirror, and a half-hour of reading and experimenting later, and I'd less than a hundred lines of Perl churning out plays in the amount of time it takes to update the start and end coords. I never would have thought it would be this easy. Now I need to do all the writeups, and I should have it up by tomorrow.

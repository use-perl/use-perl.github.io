Make the Computer Work

cwest on 2003-06-13T17:21:00

Take a break, relax, let the computer do the hard work. That's what they're there for, right? Nat and Jarkko bring to light someting about Perl that I try to forget each day. Yeah, I shudder. The truth is though, I don't care that much. Perl's print function is a system resource hog compared to C's write(), but so what?

Would computer be getting bigger, faster, more memorable (so to speak), even without bloated software like Perl? Yes, IMHO. So go ahead, use and abuse the computer. They don't care!

Of course, there are some good and bad ways to use Perl if you want performance, but often we don't think about performance. I wonder if we should...

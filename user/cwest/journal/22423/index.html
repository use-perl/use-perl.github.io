Data-Message Released

cwest on 2004-12-23T02:04:15

Data-Message is working its way through the CPAN mirrors. It makes use of Email-Simple for RFC2822 compliant message handling. It also makes grouchy mode and header folding optional within the constructor.

Now you can harness the power and speed of Email-Simple for parsing your HTTP responses. Woo. And also Hoo.

In other news: updates to Email-MessageID make it pass tests on Windows and make the uniqueness of IDs more probable, and version 1.45 of Email-Send includes a working version of Email::Send::SMTP. Version 1.44 of the distribution does not. Bummer.

More updates later.

As Theory mentioned, the macosx@perl list seems to be the place for camelbones questions, although there do seem to be a few users around here.

As for compiling, you just need the source, which comes as a Project Builder project. The instructions that Sherm gave on the list seem fairly simple. If you want to compile the framework for redistribution in the application bundle, you need to also follow these instructions.

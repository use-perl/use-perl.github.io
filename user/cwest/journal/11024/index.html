poest Initial Release

cwest on 2003-03-13T23:39:55

The first, pre-pre-super-developmental-expiramental-alpha version of poest made it to the CPAN yesterday. The goal is to get people who might be interested in writing a plugin for a Perl and POE based SMTP server to eye up the documentation and the ideas I have. I spent quite a while writing documentation for configuration classes and plugin classes. Currently, all the SMTP server can do is send a banner, demand hostnames in the 'HELO' command, and handle 'HELO' and 'QUIT'. Soon it will do more, like accept a message, or even logging!

For those interested in the name. It's pronounced "post", and the "e" is there because of POE. The idea is that the post delivers the mail, and poest delivers the email. I like it, so there!

Read the docs, or better you can run it, or even better you can read the source code! Please, whatever you do, do send me comments of any kind, with the exepction of those based upon "dude, you're crazy!"

For your perusal, poest-0.20030312.

AuthType Complicated

cwest on 2002-12-12T04:01:36

Why designer types don't like Apache's built-in Basic Auth I'll never know. In any case, I decided to experiment with Apache::AuthCookie. This is a pretty cool module. You set up a login script that's all pretty and it still uses Apache's authorization/authentication phases to make sure users are where they're allowed to be. The catch is, no login box is ever popped up by the web browser. Novel.

And with that I bid fair well.

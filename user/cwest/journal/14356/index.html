Web Services

cwest on 2003-08-28T00:29:38

I happen to believe that web services are a good middle-tier for any large application that may have many end users. Without web services you might just have a set of libraries implementing the M and C of MVC, then a single V plastered right on top of that. Am I crazy? Is it too much to build systems as web services (say, as an underpinning to more than one web site (end user))? I know I've just used about every buzzword of the last six months, but I'm actually serious. What do you think?

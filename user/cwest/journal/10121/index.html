Shaping my Open Source World

cwest on 2003-01-22T16:01:22

Over the years I've used lots and lots of tools to get my jobs done. For databases I used DBI, Tangram, now Class::DBI. Web application frameworks have evolved from CGI.pm, to mod_perl handlers, (X)SSI, Apache::Template, and now I'm leaning toward HTML::Mason. For templating I've tried home grown, HTML::Template, Text::Template, and now Template Toolkit.

What I'm saying is, my tools are evolving. I think this is a good thing, I've used a lot of them and now I can have a fairly static "bag of tricks" to pull from. Hmm... I think that might be a good talk idea. Every developer should know their bag of tricks.

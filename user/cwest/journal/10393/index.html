No Holy War Beyond This Point

cwest on 2003-02-04T21:30:50

triv and I are working on a redesign of the internal libraries of DynDNS. In doing so, we're looking at all existing options, as well as the roll-your-own option to find the one that fits best for our uses. As developers we both come into this with opinions, battle scars, and religion about how we want our software to work. At one point we wasted nearly half an hour talking about doing the same thing two ways, only to come to the end and say we don't care if it's possible to do it both ways.

It's difficult to be open, but doing so relieves a lot of problems.

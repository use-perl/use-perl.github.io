How to wankishly irritate module authors ...

Note to all readers: when people send me bug reports I generally respond quite quickly. If the report is accurate, I've been known to release new versions of modules to CPAN in under an hour, I kid you not.

However, posting sort-of bug reports (sans code, sans "here's the error or bad output I got") in a whiny journal entry is remarkably ineffective at getting things fixed.

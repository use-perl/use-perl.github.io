egg

mdxi on 2003-11-25T00:33:03

I've finished all the things I could think of for egg, which means it's time for a release. It's a little script that generates XHTML from a source file containing not-XHTML in a standalone manner. Yeah, there's a million things that do this already but itch->scratch and all that. At least it doesn't read mail. And there is documentation.

Actually it does a tiny bit of automation (note datestamps on sample output) as well as straight translation, and it was , of course, fun to bang out and frees up time for me down the road. So enjoy if it's useful.

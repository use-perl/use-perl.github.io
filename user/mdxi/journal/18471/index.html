Rescued a Wyse-60 from the construction dumpster at work yesterday. It hooked right up to the Netra with the supplied RJ45-to-DB25 adapter and worked mostly flawlessly.

Notes: Under Debian, either set the term's personality to vt100 or install the ncurses-term package. If you have it hooked up to the LOM port, be sure you drop to LOM before powering off or it'll drop to NVRAM, halting the OS. Doing a kernel patch at 9600baud takes forEVER.

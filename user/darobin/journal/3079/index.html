WxUp status

darobin on 2002-02-23T23:19:50

I've been working a few hours on WxUp today and I'm very close to a first release. It has almost all displays and some are linked to the SOAP bit so that I can actually get the list of my journal entries.

Unfortunately I've got some work-work to do tomorrow and some more urgent open source things to attend to so it won't be finished tomorrow as I'd hoped.

It also still has some rough edges that need polishing, notably exception catching to avoid seeing the app die whenever something goes wrong, configuration packaging for distribution, and caching (so that it doesn't bang like mad on use Perl whenever it starts up). One cool thing is that Wx's HTML control appears to be rather good so that I might be able to just feed it HTML for journal entries originally in HTML.

Oh, and despite the fact that some features need serious polishing, it nevertheless already boasts a splash screen and an about dialog ;-)

Smart sidebars

darobin on 2002-04-02T18:25:00

This is something I've had for a while but forgot to post here. Seeing the despair of fellow Konqueror to Mozilla switchers (and my own) I decided to remedy Mozilla's lack of customizable schemes with some sidebar components.

If you're running Mozilla, you can go get a CPAN search sidebar and a m-w.com sidebar.

Mozilla doesn't always see immediately that they were added, so you might want to use the "Tabs" menu of the sidebars to deactivate then reactivate them to make them visible.

They aren't very good-looking right now, but when I get some time I'll add a little light style.

Comments welcome!

Get some SAX

darobin on 2002-01-22T23:01:27

Mostly because of my own laziness, the PerlSAX2 specs were previously only available through the WebCVS interface at SourceForge, needless to say with addresses that either required bookmarks or a solid memory for cryptic random arangments of alphanums and stranger chars.

Now, thanks to Ilya, the situation is far better :-) The docs (presently it's only the two chapters of the spec, but this will be expanded) now live at http://perl-xml.sf.net/sax/. How much easier has it ever been to get SAX?

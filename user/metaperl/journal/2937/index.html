Class::MakeMethods rocks the house

metaperl on 2002-02-18T14:38:48

Wow, this module is a complete tour-de-force. If the flexibility and power of Perl objects is one of Perl's strengths then just imagine how awesome this module is for providing concise yet comprehense and flexible access to Perl method making.

I created a sig for myself just because of this module:

$terrence->{brannon} = Class::MakeMethods -> hacker (type => 'just another');

Leopard: Spaces

Beatnik on 2007-10-31T22:12:10

As on GNOME and KDE, Leopard now has virtual desktops. One of the features I couldn't easily find was the "Move application to different virtual desktop" function. I didn't want to configure every application in advance but just wanted to move it around ad-hoc. It took me a few minutes but I figured it out. Press F8 (or whatever key you defined to show the Spaces) and drag the application to a different space. It would be easy if you'd have some kind of menu or dock function that would do the move.

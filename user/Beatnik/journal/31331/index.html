New toy: fingerprint USB disk

Beatnik on 2006-10-16T18:22:25

My latest gadget: the Sandisk Cruzer Profile. It has a really cool fingerprint reader: It'll decrypt an encrypted partition on the fly once you slide one of your fingertips over the reader. It can also handle some kind of profile integration with application that need you to enter a password (although I haven't been able to use it in a browser window). Downside: I can't seem to get it working under Linux.

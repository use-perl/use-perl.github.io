Cisco 7200 emulating

Beatnik on 2007-07-06T09:20:33

Apparently, Dynamips is a GPL licensed Cisco router emulator (kinda like a virtual machine but then for Cisco images). It boots up using the existing Cisco images (which might be tricky to acquire if you're not registered). You can also look at Dynagen, which can generate all the necessary INI files and look at DynaGUI for a Python based graphical network map of your setup.

Lesson of the day: Kickstart - reloaded

Beatnik on 2004-07-28T14:24:48

We have a dedicated machine that handles the network based installation. For the RedHat based installations (RH, Fedora, RHEL3 etc), we use kickstart. If you plan on mixing several Red Hat based distributions, copy the initrd.img and the vmlinuz file from the CD and point to those files in the configuration file.

The error I got when using the RH9 vmlinuz on RHEL3 was

* after mountStage2, rc is -1
* not the right one

Not adding the initrd.img you need will probably result in the network drivers not being found...

RH users beware!

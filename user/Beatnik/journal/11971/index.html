Spam bounces

Beatnik on 2003-05-02T13:40:47

For the past month or so, I've been getting bounces from spam. I feel sorry for the people who get mail from those fake users on my domain but there is nothing I can do aboot it. However I've managed to discover SOME pattern in those messages. Apparently all mails are send using 'microsoft.com' (unresolved) and a semi random IP. I added a filter to automagically delete all mails containing 'microsoft.com' in the message body. Tough luck if I ever get a mail from Microsoft. I'm sure it won't be all that important.

Padre 0.16 released

gabor on 2008-11-10T07:05:54

In the name of the Padre development team I am proud to announce the release of version 0.16 of Padre, the Perl IDE.

Having more than five commiters is already a team, right?

As usual, the full list of changes is in Changes

The highlights:

  • Change windowing system to AUI providing
    Split view, Tab reordering and Tab close button.
  • Localization and German translation.
  • Change back the new-file hot-key to be Ctr-N again.
  • Add experimental (and optional) PPI based Perl5 syntax highlighting to provide 5.10 highlighting as well.
  • Tab/space Upper/Lower case conversion menu items.
  • New icons on the tool-bar adding undo/redo/cut/cop/paste/select

Known installation issue:
If you already have a plug-in installed and Padre 0.15 came with a built-in plug-in, test 82-plugin-manager will fail. This is a bug in the test.

As usual, you can install it directly from CPAN.

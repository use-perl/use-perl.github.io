perldoc.perl.org wish list

gabor on 2008-12-21T05:11:34

Here is my wish list for perldoc.perl.org

  1. Be able to type in for, foreach, while, etc. or even $_ or $a and get a link to a good explanation.

    For this the X<> tags, added in 5.10, need to be indexed.

  2. Have localized version of the site.

    Eg. fr.perldoc.perl.org (future link) should have the same thing just with the French documentation based on POD2::FR and it.perldoc.perl.org (future link) should have the Italian version based on POD2::IT, with each site linking to the other translation sites.

  3. Get access to the source code of perldoc.perl.org so I can reuse it and/or implement some of the above myself.

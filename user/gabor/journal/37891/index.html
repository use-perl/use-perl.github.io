vi mode in Padre, now for real :-)

gabor on 2008-11-16T13:19:53

So the other day my happiness over the command line part of vi was spoiled by Aristotle but not any more.

I hope.

I just added an experimental vi-mode to the Padre source code. That means starting from the next release (or earlier if you care to check out from SVN) you can suffer with me all the things that are still missing...

Basically I was aiming to have a proof of concept for the 3 major modes of vi (navigation, insert and command). Moving between them and some minimal set of keys to support.

For now it is so minimal I still have to switch back to normal Padre mode if I want to write something but it already works.

So now you vim users, you'll have to find other excuses for not trying Padre.

(/me goes hiding as Alias is surely will try to shoot him for adding bloat to Padre)

My only hope is that basically all the vi-related code is loaded only if vi-mode is enabled so those who don't use it, will never see it.

I'm not concerned...

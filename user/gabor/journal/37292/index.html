Padre 0.06 released

gabor on 2008-08-28T13:07:26

I have just uploaded v0.06 of Padre to CPAN.

Instructions on how to download and install are posted (and should be posted) here

Padre - Perl Application Development and Refactoring Environment is going to be an IDE for Perl written in Perl using wxPerl.

While it is still in early stages you can already use it and help by giving feedback or by sending patches or writing plugins.

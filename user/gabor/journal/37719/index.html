Padre 0.12 released (0.11 was broken)

gabor on 2008-10-23T14:03:30

After taking a month break I am glad to announce the release of Padre 0.12.
There are many changes as listed in the Changes file

The major outline is

  • Tons of code improvements by Adam Kennedy
  • Module::Starter integration
  • Providing call-tip for all the Perl keywords using a YAML file

You can download and install it from a nearby CPAN mirror. More detailed instructions can be found on the Download page

Windows users might want to download the latest Strawberry Perl release (5.10.0.3) as it already supports Wx installation without the extra manual work earlier releases required.

Comments, bug reports as usual on the Padre bug tracking system or on our mailing list.

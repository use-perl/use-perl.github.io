MIME::Base64

Bernhard on 2006-10-12T21:42:46

After starting on Plumhead, aka PlumHeaded Parakeet, I stumbled on base64 encoded strings embedded in the XML emitted by phc http://phpcompiler.org.

The plan is to create Parrot Intermediate Representation from XML. So I became sidetracked into making a MIME::Base64 module for PIR. As usual it took longer than expected, even though I found a nice german tutorial, So an incomplete implementation is now at . The tests I stole from the Perl5 module MIME::Base64. Thanks to Will Coleda for compilers/json, which helps in setting up the test cases.

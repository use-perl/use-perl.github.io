HQ9+ left the nest

Bernhard on 2009-01-12T21:58:25

HQ9+, , is a little language the makes a few things easy, and almost all other things impossible.

There is an implementation of HQ9+ for Parrot that is based on the Parrot Compiler Toolkit. For getting my feet wet with git, it moved the source code from Parrot's svn repository to . The migration went very smoothly and git made a good initial impression on me. The instructions on github.com were very helpful. Details on the migration are on .

In this migration I simply discarded the svn commit history. For moving other languages to git, it would be handy to have a complete and current git mirror of Parrot's svn repository.

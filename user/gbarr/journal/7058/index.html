"CPAN is Useless"

bart on 2002-08-16T23:22:39

Well to all those who think the new search site is "useless", sorry not my problem go somewhere else.

Tsk. Touchy.

In case you hadn't noticed: that "CPAN is Useless" I quoted is just the title of a lightning talk. I'm not saying <search.cpan.org> is crap, honestly, it's great. It is. But I'm familiar with CPAN. This particular talk discussed the difficulties for first time users, which we all here, are not. So we are pretty poor judges.

What I got out of that talk, was that the main complaint is that it's too damn hard to distinguish a link to download the package from all those links that merely display the docs. And that is something that has not been addressed. Perhaps it just doesn't matter to you (all).

What is so horrible about the word "download" that it seems to be a taboo to put it, literally, on a page? Because this is what I think first time users will do: hit "ctrl-F", type in "download", hit "enter", and let the browser search for the proper download link. It will find nothing.

Test Reports from an Alternate Universe

chromatic on 2009-03-30T19:24:58

Allison asked me yesterday if a Parrot 1.0 failing CPAN Testers Report made sense. I had to say no.

If you read Slaven's report, you see that his installed version of TAP::Harness understands neither the verbosity nor the jobs flags. He's running Perl 5.10, released in December 2007.

The problem is that he's apparently using a beta version of TAP::Harness, as the first stable release (3.00, released in November 2007, before 5.10) understood at least the verbosity flag.

In other words, as best as I can determine, this smoker client represents a Perl 5.10 installation with unupgraded beta modules installed. I suggested to Allison that we ignore the report, as it's a weird configuration that only people who know what they're doing would ever encounter. I expect those people to be able to diagnose and fix these problems on their own.

Musing about a wireless file-sharing appliance

dws on 2004-02-18T19:51:35

For ad hoc geek meetings where wireless isn't available, it would be really handy to have a portable wireless file sharing appliance. Something like one of these, running Linux or FreeBSD. Putting the hardware together would be no challenge--I already have Linux running on a C3-based mini-itx board. It'd cost about $400 USD. The trick, I suspect, would be in configuring the wireless, getting a DHCP server set up, dealing with Samba config, CVS, etc. Lots of little details. I'd be surprised if someone hasn't already worked through them.

Having something like this would beat the heck out of carrying around a router and a bunch of cables.

Alien::CodePress

ash on 2007-10-10T09:43:39

I've just released a new module: Alien::CodePress.

I've been using CodePress to syntax highlight code in my HTML-generated POD documentation. It's very nice. Since I didn't want to ship it with my code, I created a Alien module for it.

Download the module from the Alien::CodePress home at CPAN.
or see more info about CodePress on their website.

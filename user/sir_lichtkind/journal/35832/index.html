new features in Kephra

sir_lichtkind on 2008-03-04T14:40:44

recently i writing on Kephra again. partly on the release tools, automating the cpan and the windistro (hball) releases. I had already a tool for that but now i clean things up, extend, just want to learn and write good production code for own robust tools.

The other thing was a feature request that i got from my first full time user: reneeb++ which motivited me to add file change notify support. by the way i added also autosave, which i wanted to have for long time. Notify dialog isn't fully ready yet and both feature are set off by default. (see changes file for details how to turn them on). it will be available after version 0.3.6.8.

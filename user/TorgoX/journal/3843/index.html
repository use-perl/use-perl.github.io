I occasionally hear of problems with donating old hardware because a box isn't that much use without an OS to run it... and chances are the old box ran Windows. So you'd have to transfer the Windows licence (which, as I understand it, the EULA forbids you to do, but you can't buy any version of Windows anymore that'll run on a 486) and delete the Windows files from the backup CD... or install something such as Linux or *BSD* on it.
Just something to think about.

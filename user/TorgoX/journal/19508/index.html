The RTF Recognition Inside Me

TorgoX on 2004-06-26T06:20:26

Dear Log,

Last night, I puttered around some and wrote some contributions to the upcoming MS-Word Hacks book that O'Reilly is doing; mine are mostly from an RTF/Perl point-of-view, and which I hope will push my RTF book. (Next step: profit!) I came up with some neat tricks that turned out nice. I also spent a hew hours noodling around with Algorithm::Diff to write a differ that outputs RTF, and it ended up working fine, but I eventually decided that I hadn't done much more than just recapitulating MSWord's Compare Documents feature. Ahwell, gotta reinvent the wheel some time.

I also dug up some macros I've written over the years and sent them off to the book's main author so he can see if he likes any of them.

I finished reading Pattern Recognition, and then read The Killer Inside Me in one sitting, something I don't often do. Both books were quite good.

Someone brought back the decent weather here, so I am no longer being roasted alive. I'm leaving that to the Juneau people.

I also massaged some code so that my multingual calendar generator now has the months and days-of-the-week for Jicarilla Apache.

I did dishes while the cat berated me for the fact that I am not a giant fillet of halibut.

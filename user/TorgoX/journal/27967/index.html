Zooming in emacs

TorgoX on 2005-12-13T23:17:02

Dear Log,

Because I wanted my emacs to support browser-style font zooming, with something at least vaguely like the bindings I'm used to with Firefox: 

(require 'view)
(require 'zoom-frm)

; mouse-wheel up and down
(global-set-key [C-mouse-4]  'zoom-frm-out)
(global-set-key [C-mouse-5]  'zoom-frm-in)

; keypad plus, minus, and 0
(global-set-key [C-kp-add]      'zoom-frm-in)
(global-set-key [C-kp-subtract] 'zoom-frm-out)
(global-set-key [C-kp-insert]   'zoom-frm-unzoom)

(Needs zoom-frm.el)

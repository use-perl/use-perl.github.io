Dear Log,

Today I pretty much finished the code and docs for the Pod mini-webserver -- which I'm considering naming one of: Pod::Daemon, Pod::HTTP, Pod::HTTPD, or Pod::Webserver.

Maybe Pod::HTTPD would be best, and then the stub executable can be called "podhttpd".

I also gritted my teeth and wrote an outline for the Pod::Simple::HTML* finishing up the docs. I hate writing docs. Or rather, I obviously don't mind writing, but I hate having to write short-but-complete docs at a rate of anything less than a module per week. But in this case, I think I'll give complete if terse per-method docs, and then describe two or three common use-cases in the Description and Synopsis sections, and just tell people to email me if they need to know anything else.

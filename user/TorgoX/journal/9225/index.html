Pod::Perldoc

TorgoX on 2002-12-02T05:42:52

Dear All,

New Perldoc! It's v3.07, now entering CPAN. It fixes an obscure and probably never-run code typo to Pod::Perldoc::ToMan; adds a sentence more of docs to Pod::Perldoc::ToRtf; and makes perldoc -otk manage to limp along a bit better with the older Tk::Pod version, I think. Upgrade now and avoid the rush later!

I don't think it'll run under 5.005, but I might get around to backporting to it. Its reliance on 5.6 features is probably just incidental.

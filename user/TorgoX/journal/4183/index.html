TPJ / Sys Admin

TorgoX on 2002-04-14T07:28:40

Dear Log,

I got my copy of Sys Admin magazine in the mail today; it's the 10th anniversary issue. In the introduction on page 6, the chief editor says that the current subscribership is 33,000 -- and that's not counting newsstand sales. That means that every third month, 33,000 people get The Perl Journal. I'm quite surprised.

I just wish that there were more space for more Perl articles. The current TPJ section is pretty small, and I feel a bit bad about the fact that by writing, I'm taking up a scarce resource.

More emacs dired mode key-fun

TorgoX on 2005-05-19T10:16:28

Dear Log,

So that emacs dired mode understands my habit of hitting f5 or control-r to refresh directory listings: 

(add-hook 'dired-mode-hook
               '(lambda ()
                  (define-key dired-mode-map [f2]   'dired-do-rename)
                  (define-key dired-mode-map "\C-r" 'revert-buffer)
                  (define-key dired-mode-map [f5]   'revert-buffer)
))

Dear Log,

I spent all day doing stuff for my other job, as a techie for various Native Language preservation projects. First off, fiddling with various real-time acoustic analysis programs -- I really just need something that's a glorified tuner. A half dozen programs came close, but not quite. But in the end, I think I found something.

Then it was a mind-crushing several hours in Fontographer, doing font hacking on Cherokee fonts. I never cease to be amazed at how capricious and weird Fontographer is. It's like LaTeX: it doesn't have to work really well, because what'll you do, use something else? Hah!

I spent forever chasing a problem that turned out to be something to do with the em-square settings. Kookoo nutty font formats!

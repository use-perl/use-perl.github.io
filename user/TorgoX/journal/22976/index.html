TorgoX on 2005-02-02T01:54:37

Dear Log,

I am endlessly amused that when I try to use "ТРАНСОПЕЙ" (9 Unicode Cyrillic characters) as a post topic here, it is alternately rejected as an empty topic, or accepted but truncated at about the 8th character, ending with "&#1045;&#10" [sic]. If I add a period to that, giving "ТРАНСОПЕЙ.", that suddenly makes it non-blank and it is accepted -- except that an intermediate edit/preview phase will corrupt it ("...&#10") again.

And forget a full "ТРАНСОПЕЙВАНИЙ" (14 chars), it always gets truncated.

It's oddly reminiscent of old "8.3" naming, except this time in Cyrillic.

Whereas "Antidisestablishmentarianism or undisestablishmentarianism" (58 US-ASCII characters) is a fine topic, not too long or empty.

Man, programming is super-hard.

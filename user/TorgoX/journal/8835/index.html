Dear Log,

Last night I finally released a new HTML-Format version (containing HTML::FormatPS, HTML::FormatRTF, and HTML::FormatText, which render HTML as Postscript, RTF, and plaintext). They aren't everything you might want them to be -- they don't understand tables or inlined images -- but they're rather a step up from `lynx --dump`.

Today I worked a bit on Pod::Simple and on Perldoc 3.

Perldoc 3 is almost ready to show off. I think I'll release it ASAP, and then let people ruminate on it as I go work on Secret Project R.

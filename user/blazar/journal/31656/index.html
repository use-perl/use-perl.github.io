The Xmas tree by my side...

blazar on 2006-11-19T13:04:50

Ok, so I didn't want to talk about my disease and about the therapy too much, but it seems I end up doing so quite often...

So what? Well, I've just started this new chemotherapy cycle, and I've noticed that in this new hospital some things are different than in the previous one. For example, there they used to give the various drugs (and hydration) sequentially, while here I'm receiving (some of) them in parallel. So, I don't know which is the English name for that thing intravenous drip devices hang from, but I can tell for sure that it looks like a Christmas tree now. Hey! I even have two volumetric pumps!! With all those tubes eventually reaching my Port-a-Cath.

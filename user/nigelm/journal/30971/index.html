Theatrical Catalyst

nigelm on 2006-09-12T15:56:57

After a very very long drawn out porting process the backstage management app for the Joseph Rowntree Theatre finally went live over the weekend.

Unfortunately the stuff you can see there is only the static website, although all the show and presenting company information is generated off the bakcstage db.

The previous version was Class::DBI and Maypole based, and although very useful, had a number of limitations, and a slight flakiness - in that tweaks always seemed to have a DABWIM (Do Anything But What I Mean) effect. The Catalyst/DBIC conversion is much cleaner and better architected IMHO.

We're looking at moving the website out onto another box (it currently runs on a work spare) with the backstage management stuff - if that happens the main website may also start doing more dynamic things too...

File::SAUCE 0.1 released

LTjake on 2003-03-21T12:18:31

I've uploaded File::SAUCE to my CPAN directory.

What is SAUCE you ask?

SAUCE is a universal process to incorporate a full description for any type of file. The most outstanding aspect of this concept is that you have access to the complete file name, the file's title, the creation date, the creator of the file, the group that the creator is employed by, and much, much, more.

More info can be found on acid.org's sauce page.

In essence it's oldschool metadata primarily used in the ANSI and ASCII "art scene."

Now that I have a basis for your question, I think this will be sufficient...

Typically I do this:

use CGI qw/:all/;
use DBI;
use DBD::Oracle;
I program using a state engine, and hidden tags...
sub action()
{
  my ( $act ) = @_;

  if ( $act eq '' )
  {
    &present_menu();
  }
  elsif ( $act =~ m/^ed/ )
  {
    # get auxillary parms for this action...
    &do_edit();
  }
  # etc, etc...
}

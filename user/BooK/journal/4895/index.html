Games::Golf 0.14

BooK on 2002-05-14T08:56:26

More than one and a half month after the release of Games::Golf 0.13, we made use of the recent long French week-end to make a new release of our pet module (roughly one hundred cvs commits in three days). The official announcement will be done tonight.

It's still lacking a lot of things (and won't be uploaded to CPAN unil later today), and the documentation is very inaccurate in some places, but we feel it has grown enough to be announced and shown to the Perl golf community.

We have big hopes for this module, and particularly in the concept of .glf files. These files will hold the code for the test script, as well as the rules of the games as (guess what?) POD comments. In the future, the player script shipped with our module will integrate with the PGAS architecture so as to allow auto-submit, fetching new updated versions of the test script, and so on.

Oh, and it should work on Unix and Windows.

Back from Summer Vacation

jonasbn on 2003-07-21T13:20:11

This journal entry covers the last 2 weeks of my Perl activities.

I responded to a mail from the YAPC::Europe committee. It was not easy and required several calculations and assumptions, YAPC::Europe is more than a year away and we still have no idea whether it will be in Belfast or Copenhagen. But luckily we have some experiences from organizing the Scandinavian Perl Workshop in April.

DKUUG has agreed on co-hosting the YAPC::Europe 2004 if it comes to Denmark.

Well now the mail has been sent now I can only hope for the best (eventhough none of the members of the Copenhagen Perl Mongers would mind going to Belfast AFAIK, including myself).

For the programming part I have been doing a lot of cleaning up, not as much as I had hoped for, but I am on summer vacation and the weather have been great and I have spent time with a lot of my friends.

- Games::Bingo::Bot have been updated (0.02) - POE::Component::IRC::Simple have been spawned (0.01) - XML::Conf has been prepared for release (0.03) - Games::Bingo::Print separated out (0.01) - Games::Bingo cleaned up (0.06) So all of these will reach CPAN shortly.

Some of my larger projects are showly taking shape, parts will probably see the CPAN sometime in the future.

- VB-POD parser (I will be using this at work) - Codebase (I might have to find a new name for this project) - YAPC Registration software (This is a very simple prototype)

I am also to create a prototype for my girlfriends fathers shop and my friend working at RUC have asked me and our common friend to create a homepage for a project he is running.

The shop project until now gone quite well I have almost finished the specification and as soon as it is signed of I can start implementing the software.

The university project is quite simple and I have only received and outline for what it is supposed to look like, it does not seems a huge task.

The YAPC registration software is an idea for some component add-ons to the software we used for the registration with the Scandinavian Perl Workshop. Hopefully Paris.pm will release parts of their software so we can make a common application for use by all hosts of YAPCs (and other workshops). I am thinking about doing a BOF on this in Paris.

So I have plenty to do - I could do with some more vacation it seems...

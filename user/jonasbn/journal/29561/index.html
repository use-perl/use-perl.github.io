Back on Komodo

jonasbn on 2006-05-09T07:41:55

I bought my Komodo IDE license the other day, I am please to be using Komodo again, the triel period was a positive experience. Still there are features I miss and I think are lacking:

  • Detachment of editor windows, using multiple monitors while working seems not to be a benefit using Komodo
  • Vertical split of editor window instead of horisontal splitting, I keep to a width of 80 chars and therefor have plenty of sideways, and programs tend to grow in the length not in the width (like programmers)
  • I like the project organisation a lot, but I think the search window needs a search project feature for searching exclusively in a logical group of files
  • I miss the possibility of moving block of text easily like in BBEdit, this might be in there somewhere I just have not located it yet


More feedback will follow, now I just need to put it in a mail to ActiveState...

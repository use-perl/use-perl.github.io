PDFLib and OS X

jonasbn on 2003-05-04T16:54:17

In order to get my Games::Bingo on the way I need to reinstall PDFLib after my upgrade to OS X 10.2.

I installed PDFLib via FinkCommander, but apparently this distribution does not have any of the integrations towards Perl, Java and PHP enabled - so it is somewhat useless, since the CPAN module require this to be enabled - but FinkCommander does leave the downloaded source in the /sw/src directory.

After having tried several builds without success I turned to google and apparently the configure file does not work under OS X 10.2, you have to download a new version.

After a succesful build and installation I could install PDFLib from CPAN and now I can move on with my Games::Bingo project.

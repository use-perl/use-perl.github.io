Tie::Tools 1.05

jonasbn on 2003-12-08T21:50:42

Tonight I sat down wanting to get some of my Games::Bingo stuff out of the way, including some of the helper modules (new to CPAN).

And as written a long time ago I volunteered to maintaining XML::Conf - in order to get to know the module, I have started using it for some of the Games::Bingo stuff.

I have done a lot of work, but I cannot seem to get my first release away for this module (and several others).

I stranded in XML::Conf, since it requires Tie::Tools and Tie::Tools was giving me a hard time.

So I simply made a decision to do it right, so I started by importing Tie::Tools into the CVS server I use (courtesy of Copenhagen Perl Mongers), I then structured some of the stuff a bit more logically (to my taste) and finally I uploaded Tie::Tools 1.05.

Here is the changes from the Changes file:

1.05 Mon Dec 8 21:36:57 CET 2003

  • maintenance taken over by jonasbn
  • 1.03 was the latest working release I could find on CPAN, 1.04 is there, but it seems empty.
  • renamed manifest to MANIFEST, elimnating build warning
  • added module Tie::Tools and single point of entry for Tie::Tools
  • moved $VERSION for Tie::Tools from Tie::Func to Tie::Tools
  • moved readme content to newly added TODO
  • fixed breaking test (issuing warning instead of death sentence)


Tomorrow night I will commence with some of my other modules...

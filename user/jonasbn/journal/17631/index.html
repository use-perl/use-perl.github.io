Module names and new promises

jonasbn on 2004-02-26T08:00:09

Following the advice given by brian_de_foy in his comment to my journal entry on 2nd. of February, I have renamed my IRC module to IRC::Bot::Dumb.

I have a need for programming simple bots, so I have generalized the IRC code from Games::Bingo::Bot 0.01, I think I have mentioned this earlier in my journal.

This general code is used in, Games::Bingo::Bot 0.02 of course, but will also be put to use in other places, like my upcoming IRC::Poll, which is soon to be released, together with Games::Bingo::Bot 0.02.

IRC::Poll will be a simple module to make polls on IRC.

I have several other ideas for IRC::* modules (which we need on the Copenhagen Perl Mongers) channel #cph.pm on irc.perl.org.

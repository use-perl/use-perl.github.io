Subversion hosting and clients

jonasbn on 2006-09-09T14:58:31

Some time ago I contacted my hosting company and requested subversion hosting via a subversion hotel, I would like to host it myself, but I do not have the time to take care of the details. After about a week they responded, they had setup a solution for me and I got the required users created again via the hosting company.

I have however not been able to nothing but browse the repository via a webbrowser, suspecting my svn-client I attempted to update to the ssl version using, fink, but I could not get it to build, so I looked around for alternatives and I located Martin Ott, the client on his site however did not work for, so I suspected it not being able to do SSL, but I noticed an interesting blog entry: Shelf Cloud thing and until now it has actually worked out of the box. I also located an SSL enabled client for OSX.http://www.codingmonkeys.de/mbo/articles/2006/08/24/subversion-hosting-offered-by-shelf-cloud>, so I set up an hotel with this Shelf Cloud thing and until now it has actually worked out of the box. I also located an SSL enabled client for OSX.
So currently everything is working fine, with Shelf Cloud I can have more than one repository, I can create and control users myself. It is a but expensive, but I just have to get my clients to pay :)

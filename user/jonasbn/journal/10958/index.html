Spring Cleaning the HDD

jonasbn on 2003-03-09T12:12:22

I was going through some of all the .sits, .dmgs and tar-balls. I have gathered on my harddrive. It is amazing the number of files you can have lying around which you do not remember what is exactly. This was what survivied:

  • Amazon_Webservice_kit.zip
    The is a renamed version of the 'kit.zip' from Amazon holding their webservices developer stuff.
  • BBEdit_7.0.1_Demo.dmg.bin
    The coolest editor for OS X, yes I have registered
  • CBInstaller.dmg.gz
    CamelBones Cocoa-Perl bridge/framework, I have developed several applications using the framework such as a calculator and a Bingo game :)
  • CandyBar_1.0.2.dmg.bin
    A piece of shareware I consider registering, it gives you easy control of icons.
  • CodeTekVirtualDesktop.dmg.gz
    A virtualdesktop which is more professional than space, but it is shareware so for now I still use space
  • DOCtor-2002-07-13.dmg.gz
    A M$ Word to PDF converter, I have word but I this program just because it is freeware.
  • Desktastic.sit
    A phanstatic app letting write and draw on your desktop (shareware).
  • DesktopRandomizer101.sit
    Well... random backgrounds...
  • DropScript-0.5.dmg
    The famous dropscript, I have it but I really haven't gotten around to using it.
  • Fire.app0.32.b.2.dmg
    Instant messenger program, not as cool as Trillian for windows, but it is ok
  • LimeWireOSX.zip
    Gnutella-based client it is nice to have at least one peer2peer client.
  • MacBackOrificePPC.sit
    After installing wifi at home I wanted to try to find out how easy it would be to hack my wintel 98 box
  • OpenTerminalHere.sit
    A small utility, which sounds usable, but I don't - but at the same time I don't want to throw it away
  • Perl-5.8.0-MacOSX.tar.gz
    An installation of Perl 5.8.0
  • Pine-Pico-4.33X.pkg.sit
    My favorite mail client of all times, I use Mail.app now, and use Pine as a newsreader...
  • ShellService.tar.gz
    An addition to the service menu, so you can execute text

    Shockwave, necessary in these multi-media days
  • SigMeter.sit
    A Wifi signal strength indicator
  • Space_App.tar.gz
    A freeware virtualdesktop program really cool

  • Test-Smoke-1.10.tgz
    The Test-Smoke suite for smoke-testing Perl builds (I think this has updated radically)
  • ThermoInDock.0_71.src.tgz
    A no longer supported open-source program which can tell you your CPU temperature in the dock
  • TinkerTool2.dmg.gz
    A really cool tool which can be used to control the terminal transparency among other things

    A shareware driver for my webcam
  • UnRar30MacOSX.tgz

  • Warcraft_III_Demo_Installer.bin

  • X-Chat_Aqua.0.7.0.dmg

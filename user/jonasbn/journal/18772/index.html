Module::Template::Setup 0.03 uploaded to CPAN

jonasbn on 2004-05-15T14:39:53

I have just uploaded Module::Template::Setup 0.03 to CPAN.

The update contain nothing really interesting, it just a way for me to get going again with the development.

I have added handling of licenses now. So now it is possible to choose between four different known licenses (artistic, gpl, lgpl and bsd) and then get the license details filled out or you can add you completely own license and description.

I have also added possibility of outputting a Build.PL instead of a Makefile.PL - this was a kind of proof of concent and it seems that CGI::FastTemplate can handle adding templates at all times, so this opens up for the handling of user created templates... so handling of this should be in the next release...

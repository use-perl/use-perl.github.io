fscking Comcast

geoff on 2002-02-13T15:24:25

so, I've been wondering why my static IP address hasn't been the same as the one that shows up when I surf. "is comcast using a proxy?" I ask myself. now I know.

Comcast tracks users' Web browsing

looks like comcast (my cable internet provider) has all web surfing going through a proxy so they can track my surfing habits, credit card numbers, and so on. it's not a voluntary proxy either, so I can't shut it off. I find it unbelievable that they didn't notify customers about this invasion of privacy, not that I would have done anything about it, but I think that they should have at least made everyone sign something saying they understood the implications (though it is unlikely that Comcast understands the implications themselves)

time to start looking harder at anonymizer or, better yet, find another provider. I'm about fed up with this monopoly Comcast has over my life, with their ever increasing prices, eavesdropping, and other such nonesense.

Apache 2.0.45 left behind

geoff on 2003-08-25T13:37:02

late last week, support for Apache versions older than 2.0.46 was dropped from the development version of mod_perl 2.0. libapreq (which includes the popular Apache::Request) already required 2.0.46, which made that version as good as any to make the break and clean up mod_perl's internals a bit.



so, if you plan on installing the next release of mod_perl 2.0 then you should get busy updating your httpd as well.



and fear not, mod_perl 1.0 still supports many, many different versions of Apache 1.3.

Extending and Embedding Perl

geoff on 2002-09-10T12:25:03

I preordered my copy of Extending and Embedding Perl months ago from Amazon, but I finally found it in my mailbox only yesterday.

I've only spent an hour or so going through it, but so far it's very impressive. what really got me excited was that so many of the XS examples are pulled from mod_perl - better, they are things I've spent some time pouring over already and the commentary looks to broaden my understanding even more. I also saw a few examples from other interesting modules like DBI, so I'm really anxious to find a few hours where I can really start to devour it.

looks like an essential book for my library. well done all.

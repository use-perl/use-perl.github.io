Devel::Profiler development

geoff on 2003-09-23T12:52:56

my current project has me working a bit on Devel::Profiler, a pure-Perl Devel::DProf replacement by Sam Tregar, helping along some of the ToDo items listed for the package. after a few good conversations with Sam, he kindly set up a Devel::Profiler mailing list on sourceforge. so, all you bricolage, mason, and other like-minded profilers come on over :)



of course, open source development is more circuitous than linear, and my personal Devel::Profiler hacking has taken a turn. seeing that there was no test suite for Devel::Profiler::Apache I thought I should write one before getting deep into the module. that turned into a project in itself, since Apache-Test does not have (yet) have a mechanism for running tests outside a running server. so, as soon as I get this little problem resolved, I'll be back on to Devel::Profiler...

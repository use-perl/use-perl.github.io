mod_blosxom 0.04 released

oyama on 2002-08-22T17:41:16

mod_blosxom.c is the Apache module which builds the extremely lightweight Weblog environment. mod_blosxom has inherited the function of Blosxom and can share the same data and the same template.

mod_blosxom module for Apache


  • What is Blosxom?
    • Blosxom is the very compact and portable Weblog tool written by Rael Dornfest. Blosxom is the cool script of only 61 lines written by Perl.
    What translated this Blosxom into the C language and Apache API. It's mod_blosxom.

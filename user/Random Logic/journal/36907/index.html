Moosifying logging

Random Logic on 2008-07-14T00:46:36

Moose is great, at least the parts that I've discovered so far. It gives a very clean interface for working with OO in perl.
After some playing around and discovering the main concepts behind moose and failing to find a logging role using my favorite logging system log4perl I sat down an did a little coding (it is really just a few lines) and uploaded to CPAN as MooseX::Log::Log4perl.

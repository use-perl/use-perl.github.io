Working From Home

vek on 2001-12-14T22:37:16

So today I worked from home. Working from home is nice *but* a tad annoying. Because of the anal firewalling we have to use a proprietary *windows only* VPN client if we want to get onto our network and that sucks let me tell you. So before I left work yesterday I downloaded everything I would need onto my linux laptop. I've got MySQL on it (albeit a somewhat scaled down database of course) and I've got the directory structure somewhat similar to the dev Sun box at work. Lo and behold I can develop and test just as if I was on the Sparc. Except that I'm on my sofa with my feet up. Which is nice. *grin*.

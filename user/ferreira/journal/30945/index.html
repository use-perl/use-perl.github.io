How many useless modules is possible to release in 30 days? How many of my broken modules can be improved? I have an enormous backlog and a half opportunity to do some Perl programming at last. But that's triffles - hard is to get and retain users. For example, so far the only module I wrote that is a dependency to a CPAN module is Number::Bytes::Human. (Thanks, Jonathan Rockway and his Chroniton module.)

Now that you got curious about N::B::H, it is a simple number formatter that takes a number and formats it as the -h option in ls, du and other utilities (like '0', '2K', '3G', etc.)

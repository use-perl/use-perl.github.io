dbi debugging aid

gav on 2002-09-19T15:29:41

Want to know what sql you're throwing at the database but you're binding variables. This should do the trick: 

use Text::Wrap;
my $temp = $sql;
$temp =~ s/\?/$dbh->quote($_)/e for @data;
print $cgi->pre(wrap('','',$temp));

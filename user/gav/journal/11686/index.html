Spam Spam Spam!

gav on 2003-04-16T20:07:16

For some reason the last two weeks I've been getting loads of spam that has sneaked passed Spamassassin. These have been mainly short annoying emails with image attachments or for porn sites in German. I've finally got Vipul's Razor installed, a process far easier than I expected (though it did screw up Digest::MD5). The newest version of Spamassassin and Razor2 seems to have cut down on today's false-negatives. I did get this beauty though:

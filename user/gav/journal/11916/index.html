the cost of hardware

gav on 2003-04-30T02:02:52

For a mere $250 I am now the proud owner of:

  • 2 x 80gb IBM 7200rpm hard disks
  • 24" round IDE cable
  • 128mb PC100 memory
  • 425VA UPS

The fall in the price of hardware over the last few years is scary. My slightly aging PIII-550 home server will now have a RAID-1 partition so I can back stuff up from my main server.

It will also mean I have the storage capacity to start working on my top secret project :)

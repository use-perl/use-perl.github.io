Clairvoyant observers may notice certain packs from the tiny corner of the internet are good ones to clone to avoid repacking later.

Early adopters can

git clone http://git.utsl.gen.nz/perl

It's easy to change the repository later to its eventual home possibly on cpan.perl.org, using the git 1.5+ commands:

git remote rm origin
git remote add origin git://git.perl.org/perl

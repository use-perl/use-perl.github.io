Ruby will not insert semicolons if it would be syntactically incorrect to do so:

foo =
    bar.
      select { |e| e.yah? }.
      collect { | e| e.noooo! }.
      select { |e| e.ok? }

However do tell me if you find an adequate replacement for strict.
